# 2. 南明钱币详细介绍

## 2.1 大明通宝

### 大明通宝

* **铸造时间**：南明鲁王朱以海监国时期（公元1645年—1646年）。

* **材质成分**：青铜。

* **尺寸重量**：小平直径约2.5厘米，重约3—3.5克。

* **面背特征**：

  * 圆形方孔

  * 面文"大明通宝"四字楷书直读，为国号钱

  * 光背为主

* **核心背景**：

  * 南明鲁王朱以海在浙江绍兴监国时铸，铸期极短

  * 鲁王政权存在时间短，铸量极少

  * 据《中国钱币大辞典》记载，大明通宝为南明钱币中的稀少品种

* **版别体系**：

  * **小平光背版**：背面无文

  * **小平背星版**：背面铸星纹

## 2.2 弘光通宝

### 弘光通宝

* **铸造时间**：南明福王朱由崧弘光元年（公元1645年）。

* **材质成分**：青铜。

* **尺寸重量**：

  * 小平直径：约2.5厘米，重约3—3.5克

  * 折二直径：约2.8厘米

* **面背特征**：

  * 圆形方孔

  * 面文"弘光通宝"楷书直读

  * 分小平和折二两种

  * 小平有南京版、凤阳版、桂林版、贵阳版、背星版多种版别

  * 折二钱较少见

* **核心背景**：

  * 南明福王朱由崧在南京即位后铸，铸期仅一年即清军南下攻破南京

  * 弘光通宝版别较多，反映了南明初期多局铸钱的状况

  * 据《中国钱币大辞典》记载，弘光通宝为南明铸币中较常见的品种

* **版别体系**：

  * **小平南京版**：铸于南京

  * **小平凤阳版**：铸于凤阳

  * **小平桂林版**：铸于桂林

  * **小平贵阳版**：铸于贵阳

  * **小平背星版**：背面铸星纹

  * **折二版**：较少见

## 2.3 隆武通宝

### 隆武通宝

* **铸造时间**：南明唐王朱聿键隆武年间（公元1645年—1646年）。

* **材质成分**：青铜。

* **尺寸重量**：

  * 小平直径：约2.5厘米，重约3—3.5克

  * 折二直径：约2.9厘米，重约8.88克

* **面背特征**：

  * 圆形方孔

  * 面文"隆武通宝"楷书直读

  * 有小平、折二两种

  * 铸于福建

* **核心背景**：

  * 南明唐王朱聿键在福州即位后铸于福建，铸期仅一年余

  * 唐王政权存在时间短，铸量有限

  * 据《中国钱币大辞典》记载，隆武通宝存世较少

* **版别体系**：

  * **小平版**：楷书直读，光背

  * **小平大字版**：钱文字体较大

  * **折二版**：较少见

## 2.4 永历通宝

### 永历通宝

* **铸造时间**：南明桂王朱由榔永历年间（公元1646年—1661年）。

* **材质成分**：青铜。

* **尺寸重量**：

  * 小平直径：约2.5厘米，重约3—4克

  * 折十型背上壹下分者直径4.55厘米，孔径0.9厘米，厚度0.2厘米，重22.5克

* **面背特征**：

  * 圆形方孔

  * 面文"永历通宝"楷书直读

  * 有小平、折二、折五、折十四种面值

  * 钱背铸有星月纹及多种文字

  * 铸造大小厚薄不一，版别复杂

* **核心背景**：

  * 南明桂王朱由榔在广东肇庆即位后铸

  * 是南明政权中持续时间最长的铸币，也是南明钱币中版别最为复杂的品种

  * 永历帝流亡缅甸后被俘，南明覆亡

  * 据《中国钱币大辞典》记载，永历通宝版别极为丰富

* **版别体系**：

  * **小平光背版**：背面无文

  * **小平背星月版**：背面铸星纹或月纹

  * **折二版**：背铸纪值文字

  * **折五版**：背铸纪值文字

  * **折十版**：背上"壹"下"分"，直径约4.55厘米

  * **兴朝通宝式版**：与兴朝通宝风格相近的版别
