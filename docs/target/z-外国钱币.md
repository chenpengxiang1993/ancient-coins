# 2. 外国钱币详细介绍

## 2.1 光中通宝

### 光中通宝

* **铸造时间**：越南西山朝阮惠光中年间（公元1786年—1792年）。

* **材质成分**：材质因产地而异，以青铜（铜、锡、铅合金）为主。日本钱币多为青铜质地；朝鲜钱币多为青铜质地；越南钱币有青铜和锌质两种。出土后青铜钱表面多呈青绿色锈蚀。

* **尺寸重量**：小平直径约2.4厘米，重约2.5—3.5克。

* **面背特征**：

  * 圆形方孔

  * 面文"光中通宝"四字楷书直读

  * 有汉文版和喃字版两种

  * 光背或背有纹饰

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。光中通宝为越南西山朝阮惠光中年间（公元1786年—1792年）铸行。制作流程为：先雕刻光中通宝阴文母范（含面文"光中通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。有汉文版和喃字版两种，因中越贸易频繁，部分流入中国南方地区。

* **核心背景**：

  * 越南西山朝阮惠所铸。阮惠为越南西山朝的开创者之一

  * 光中通宝为越南历史上重要铸币之一

  * 因中越贸易频繁，部分光中通宝流入中国南方地区

  * 据《中国钱币大辞典》记载，光中通宝为中国南方常见的越南流入钱币

* **版别体系**：

  * **汉文小平版**：面文汉文楷书

  * **喃字版**：面文含越南喃字

  * **背纹版**：背面有纹饰

## 2.2 景兴通宝

### 景兴通宝

* **铸造时间**：越南后黎朝景兴年间（公元1740年—1786年），铸行长达46年。

* **材质成分**：材质因产地而异，以青铜（铜、锡、铅合金）为主。日本钱币多为青铜质地；朝鲜钱币多为青铜质地；越南钱币有青铜和锌质两种。出土后青铜钱表面多呈青绿色锈蚀。

* **尺寸重量**：小平直径约2.4厘米，重约2.5—3.5克。另有多种大钱。

* **面背特征**：

  * 圆形方孔

  * 面文"景兴通宝"四字楷书直读

  * 版别极多，有小平、折二、折三、背文、光背多种

  * 部分背面铸有纪年、纪地文字

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。景兴通宝为越南后黎朝景兴年间（公元1740年—1786年）铸行，铸行长达46年。制作流程为：先雕刻景兴通宝阴文母范（含面文"景兴通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融青铜液或锌液，冷却脱范后逐枚修整。版别极多，铸量极大，大量流入中国南方。

* **核心背景**：

  * 景兴通宝为越南历史上铸行时间最长、版别最多的钱币

  * 后黎朝景兴年间长达46年，铸量极大

  * 景兴通宝大量流入中国南方，是中国境内最常见的越南古钱

  * 据《中国钱币大辞典》记载，景兴通宝版别极为复杂

* **版别体系**：

  * **小平光背版**：背面无文

  * **小平背文版**：背面铸有文字

  * **折二版**：较少见

  * **大钱版**：多种面值

## 2.3 宽永通宝

### 宽永通宝

* **铸造时间**：日本江户时代宽永年间（公元1626年—1643年始铸），此后长期铸行至明治维新。

* **材质成分**：材质因产地而异，以青铜（铜、锡、铅合金）为主。日本钱币多为青铜质地；朝鲜钱币多为青铜质地；越南钱币有青铜和锌质两种。出土后青铜钱表面多呈青绿色锈蚀。

* **尺寸重量**：小平直径约2.4厘米，重约3—3.5克。

* **面背特征**：

  * 圆形方孔

  * 面文"宽永通宝"四字楷书直读

  * 光背或背有纹饰、文字

  * 版别极多

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。宽永通宝为日本江户时代宽永年间（公元1626年—1643年始铸）铸行，此后长期铸行至明治维新。制作流程为：先雕刻宽永通宝阴文母范（含面文"宽永通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。铸量极大，持续铸造长达二百余年，大量流入中国东南沿海地区。

* **核心背景**：

  * 宽永通宝是日本历史上铸量最大、铸行时间最长的钱币

  * 自宽永年间始铸后，持续铸造长达二百余年

  * 宽永通宝大量流入中国，尤其在东南沿海地区流通广泛

  * 据《中国钱币大辞典》记载，宽永通宝是中国境内最常见的日本古钱

* **版别体系**：

  * **光背版**：背面无文

  * **背"文"版**：背面铸"文"字

  * **背"小"版**：背面铸"小"字

  * **长字版**：面文字体较长

  * **其他版**：版别极多

* **尺寸重量**：小平直径约2.4厘米，重约2.5—3.5克。

* **面背特征**：

  * 圆形方孔

  * 面文"景盛通宝"四字楷书直读

  * 光背为主

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。景盛通宝为越南西山朝阮光缵景盛年间（公元1793年—1801年）铸行。制作流程为：先雕刻景盛通宝阴文母范（含面文"景盛通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。铸量有限，流入中国南方的数量不多。
