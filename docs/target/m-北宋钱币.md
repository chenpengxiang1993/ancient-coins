# 2. 北宋钱币详细介绍

## 2.1 宋元通宝

### 宋元通宝

* **铸造时间**：宋太祖赵匡胤建隆元年（公元960年）。

* **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—70%，锡含量约5%—10%，铅含量约15%—25%。北宋铜钱铜质总体较好。青铜钱出土后表面多呈青绿色或黑漆古锈蚀，原始铸造色泽为金黄色。铁钱呈黑褐色，锈蚀后多呈红褐色。

* **尺寸重量**：

  * 直径：约2.5厘米

  * 厚度：约0.1—0.15厘米

  * 单枚重量：约3.5克

* **面背特征**：

  * 面文"宋元通宝"，四字对读，楷书含隶意

  * 文字风格仿后周周元通宝，端正规整

  * 外廓规整，好廓清晰

  * 背面有光背、背星月纹、背"日"字

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。宋元通宝为宋太祖赵匡胤建隆元年（公元960年）铸行，是北宋开国铸币。制作流程为：先雕刻宋元通宝阴文母范（含面文"宋元通宝"四字，楷书含隶意，文字风格仿后周周元通宝），外廓规整，中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。铸工精整，沿袭五代后周铸币传统，各监均有铸造，铸量大。

* **核心背景**：

  * 宋元通宝是北宋开国铸币，铸量大，据文献记载各监均有铸造

  * 面文"宋元"即"大宋开元"之意，非年号钱


  * 铸工精整，沿袭五代后周铸币传统

  * 为宋代钱币制度的开端

* **版别体系**：

  * **光背版**：背面无文无纹，最为常见

  * **背星月版**：背面铸有星纹或月纹

  * **背"日"字版**：背面铸"日"字，较少见

  * **大字版**：面文字体较大

## 2.2 太平通宝

### 太平通宝

* **铸造时间**：宋太宗赵光义太平兴国年间（公元976年—984年）。

* **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—70%，锡含量约5%—10%，铅含量约15%—25%。北宋铜钱铜质总体较好。青铜钱出土后表面多呈青绿色或黑漆古锈蚀，原始铸造色泽为金黄色。铁钱呈黑褐色，锈蚀后多呈红褐色。

* **尺寸重量**：

  * 直径：约2.4—2.5厘米

  * 厚度：约0.1厘米

  * 单枚重量：约3.5克

* **面背特征**：

  * 面文"太平通宝"，四字对读，楷书

  * 文字规整端庄

  * 外廓规整，好廓清晰

  * 背面光素无文

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。太平通宝为宋太宗赵光义太平兴国年间（公元976年—984年）铸行，是中国历史上第一种以年号为面文的流通铸币。制作流程为：先雕刻太平通宝阴文母范（含面文"太平通宝"四字，楷书），文字规整端庄，中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。铸工精整，铸量大。

* **核心背景**：

  * 太平通宝是中国历史上第一种以年号为面文的流通铸币（年号钱），开创了以年号命名钱币的惯例

  * 此后宋代每逢改元必铸新钱，成为定制

  * "太平"寓意天下太平，反映了宋初统一后的政治愿景

* **版别体系**：

  * **光背版**：背面无文，最为常见

  * **大字版**：面文字体较大

  * **阔缘版**：外廓较宽

## 2.3 淳化元宝

### 淳化元宝

* **铸造时间**：宋太宗赵光义淳化年间（公元990年—994年）。

* **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—70%，锡含量约5%—10%，铅含量约15%—25%。北宋铜钱铜质总体较好。青铜钱出土后表面多呈青绿色或黑漆古锈蚀，原始铸造色泽为金黄色。铁钱呈黑褐色，锈蚀后多呈红褐色。

* **尺寸重量**：

  * 直径：约2.4厘米

  * 厚度：约0.1厘米

  * 单枚重量：约3.5克

* **面背特征**：

  * 面文"淳化元宝"，有真书（楷书）、行书、草书三种书体

  * 三种书体均为太宗赵光义御书亲笔

  * 外廓规整

  * 背面光素无文

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。淳化元宝为宋太宗赵光义淳化年间（公元990年—994年）铸行，是中国最早的御书钱。制作流程为：先雕刻淳化元宝阴文母范（含面文"淳化元宝"四字），有真书、行书、草书三种书体，均为太宗御书亲笔，中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。三种书体分别铸行，标志着宋代对钱制度的萌芽。

* **核心背景**：

  * 淳化元宝是中国最早的御书钱，太宗以三种书体亲笔书写钱文，开创了皇帝御书钱文的先河

  * 此后宋代多位皇帝效仿御书钱文，成为宋代钱币的一大特色

  * 淳化元宝三种书体的铸行，也标志着宋代对钱制度的萌芽

* **版别体系**：

  * **真书版**：楷书面文，笔画规整

  * **行书版**：行书面文，流畅自然

  * **草书版**：草书面文，潇洒飘逸
