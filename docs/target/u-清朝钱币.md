# 2. 清朝钱币详细介绍

## 2.1 天命通宝

### 天命通宝

* **铸造时间**：后金太祖努尔哈赤天命年间（公元1616年—1626年）。

* **材质成分**：黄铜（铜锌合金），铜含量约55%—65%，锌含量约25%—35%。后金开国铸币。黄铜钱呈金黄色，表面多呈深褐色包浆。

* **尺寸重量**：

  * 汉文小平直径：约2.4厘米，重约3—4克

  * 满文小平直径：约2.7厘米，重约5—6克

* **面背特征**：

  * 圆形方孔

  * 分汉文、满文两种

  * 汉文"天命通宝"四字楷书直读

  * 满文为老满文，光背

  * 铸工粗犷

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。天命通宝为后金开国铸币，努尔哈赤统一女真各部后建立后金，建元天命时铸行。制作流程为：先雕刻天命通宝阴文母范（含面文"天命通宝"，汉文楷书或老满文），中央方孔设金属芯；合范浇铸熔融黄铜液，冷却脱范后逐枚修整。铸工粗犷，汉文版铸量少于满文版。

* **核心背景**：

  * 后金开国铸币，是清代钱币体系的开端

  * 努尔哈赤统一女真各部后建立后金，建元天命

  * 汉文版铸量少于满文版

  * 据《中国钱币大辞典》记载，天命通宝为清钱中的名誉品

* **版别体系**：

  * **汉文小平版**：面文汉文楷书，光背

  * **满文小平版**：面文老满文，光背

## 2.2 天聪通宝

### 天聪通宝

* **铸造时间**：后金太宗皇太极天聪年间（公元1627年—1636年）。

* **材质成分**：黄铜（铜锌合金），铜含量约55%—65%，锌含量约25%—35%。后金铸币。黄铜钱呈金黄色，表面多呈深褐色包浆。

* **尺寸重量**：折十型，直径约3.3厘米，重约8—10克。

* **面背特征**：

  * 圆形方孔

  * 面文满文"天聪通宝"（老满文）

  * 背面有满文纪值

  * 铸工粗犷

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。天聪通宝为后金太宗皇太极天聪年间（公元1627年—1636年）铸行，为折十型大钱。制作流程为：先雕刻天聪通宝阴文母范（含面文满文"天聪通宝"），中央方孔设金属芯；合范浇铸熔融黄铜液，冷却脱范后逐枚修整。铸工粗犷，存世较少。

* **核心背景**：

  * 后金太宗皇太极天聪年间铸，为折十型大钱

  * 天聪通宝存世较少，满文折十型为主

  * 据《中国钱币大辞典》记载，天聪通宝为后金时期重要铸币

* **版别体系**：

  * **满文折十版**：面文满文，折十型

## 2.3 顺治通宝

### 顺治通宝

* **铸造时间**：清世祖福临顺治年间（公元1644年—1661年）。

* **材质成分**：黄铜（铜锌合金），铜含量约55%—65%，锌含量约25%—35%，锡铅含量约3%—8%。清初铸币沿袭明末黄铜材质。黄铜钱呈金黄色，表面多呈深褐色至黑色包浆，不易生绿锈。

* **尺寸重量**：小平直径约2.5厘米，重约3.5—4克。

* **面背特征**：

  * 圆形方孔，内外郭规整

  * 面文"顺治通宝"四字楷书，对读

  * 背面分为五种版式（顺治五式）

  * 青铜质，铸工精良

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。顺治通宝为清入关后第一种全国统一铸币，清世祖福临顺治年间（公元1644年—1661年）铸行。制作流程为：先雕刻顺治通宝阴文母范（含面文"顺治通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融黄铜液，冷却脱范后逐枚修整。铸工精良，沿袭明末黄铜材质。"顺治五式"（仿明式、汉字纪局式、一厘式、满文纪局式、满汉文纪局式）反映了清代货币制度的完善过程。

* **核心背景**：

  * 清入关后第一种全国统一铸币

  * 顺治通宝确立了清代钱币的基本形制，"顺治五式"为后世遵循

  * 据《中国钱币大辞典》记载，顺治通宝五种版式反映了清代货币制度的完善过程

* **版别体系**：

  * **仿明式（第一式）**：光背为主，另有背星、背圈、背纪值多种。铸行于顺治元年至八年（公元1644年—1651年），仿明朝钱币风格，顺治十七年停止使用并回收

  * **汉字纪局式（第二式）**：背铸单一汉字纪局。铸行于顺治元年至十年（公元1644年—1653年），共发现十九个钱局三十一种背文：户、工、宣、临、延、云、同、蓟、原、阳、河、浙、东、荆、宁、昌、襄、福、新。其中背上工、背上新未正式发行，背右延较罕见

  * **一厘式（第三式）**：背穿右铸局名、穿左铸"一厘"二字。铸行于顺治十年至十四年（公元1653年—1657年），"一厘"意为折银一厘。共发现十七种：户一厘、工一厘、宣一厘、同一厘、蓟一厘、原一厘、阳一厘、河一厘、浙一厘、东一厘、宁一厘、昌一厘、福一厘、陕一厘、江一厘、云一厘、临一厘

  * **满文纪局式（第四式）**：背铸满文纪局，仅有宝泉（户部）、宝源（工部）两局。铸行于顺治十四年（公元1657年），各省局一律停铸，专由户部宝泉局与工部宝源局铸行

  * **满汉文纪局式（第五式）**：背铸满汉两种文字纪局。铸行于顺治十七年至十八年（公元1660年—1661年），据《清朝通志》记载有十五个地方局，实际开铸约十三个局：同（山西大同）、宁（江南江宁）、东（山东）、昌（湖广武昌）、宣（直隶宣化）、原（山西太原）、蓟（直隶蓟州）、河（河南）、临（山东临清）、浙（浙江）、陕（陕西）、江（江西）、福（福建，仅见数枚样钱）

## 2.4 康熙通宝

### 康熙通宝

* **铸造时间**：清圣祖玄烨康熙年间（公元1662年—1722年）。

* **材质成分**：黄铜（铜锌合金），铜含量约55%—65%，锌含量约25%—35%，锡铅含量约3%—8%。据《清史稿·食货志》记载，康熙钱铜质精良，"铜六铅四"。"罗汉钱"铜质尤为精纯，色泽偏亮黄。黄铜钱呈金黄色，表面多呈深褐色包浆。

* **尺寸重量**：小平直径约2.6厘米，重约4克。

* **面背特征**：

  * 圆形方孔，内外郭规整

  * 面文"康熙通宝"四字楷书，对读

  * 背面满文纪局（宝泉、宝源）或满汉文纪局

  * 青铜质，铸工精良

  * "罗汉钱"为康熙通宝中的特殊版别

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。康熙通宝为清圣祖玄烨康熙年间（公元1662年—1722年）铸行，铸行时间长达61年。制作流程为：先雕刻康熙通宝阴文母范（含面文"康熙通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融黄铜液，冷却脱范后逐枚修整。据《清史稿·食货志》记载，康熙钱铜质精良，"铜六铅四"。铸工精良，铸量极大，版别丰富。"罗汉钱"铜质尤为精纯，色泽偏亮黄。

* **核心背景**：

  * 康熙朝铸行时间长达61年，铸量极大

  * 康熙通宝版别丰富，纪局体系包括宝泉、宝源两京局（满文纪局）和二十种满汉文纪局（诗文二十品：同福临东江，宣原苏蓟昌，南河宁广浙，台桂陕云漳），另加巩局，共计二十二种

  * "罗汉钱"因传说与康熙六十寿辰有关而成为名誉品

  * 据《中国钱币大辞典》记载，康熙通宝是清代最常见的钱币之一

* **版别体系**：

  * **宝泉局版**：背满文"宝泉"（户部），铸量最大。按面文风格细分有：
    * **标准版**：面文规矩端庄，为最常见版式
    * **大字版**：面文字体较大
    * **小字版**：面文字体较小
    * **阔缘版**：外缘较宽
    * **单点通**："通"字走之底上部为单点
    * **双点通**："通"字走之底上部为双点
    * **背星月**：背面铸有星纹或月纹，较为少见

  * **宝源局版**：背满文"宝源"（工部），铸量仅次于宝泉局。亦有标准版、大字版、小字版多种细分

  * **满汉文纪局版**：背满汉文纪局，诗文二十品（同福临东江，宣原苏蓟昌，南河宁广浙，台桂陕云漳）另加巩局，共计二十一种满汉文纪局。各局对应铸地如下：同（山西大同）、福（福建福州）、临（山东临清）、东（山东济南）、江（江苏江宁）、宣（直隶宣化）、原（山西太原）、苏（江苏苏州）、蓟（直隶蓟州）、昌（湖广武昌）、南（湖南长沙）、河（河南开封）、宁（甘肃宁夏）、广（广东广州）、浙（浙江杭州）、台（福建台湾）、桂（广西桂林）、陕（陕西西安）、云（云南昆明）、漳（福建漳州）、巩（甘肃巩昌）。据《中国钱币大辞典》记载，各局铸量差异极大，其中台、巩、南、漳铸量极少

  * **罗汉钱**：据《中国钱币大辞典》记载，罗汉钱为康熙通宝中的特殊版别，具有以下特征：
    * 面文"熙"字少左侧一撇，写作"臣"上加"巳"形
    * "通"字走之底为单点
    * 铜色金黄，铜质精良，传为康熙六十年（公元1721年）圣祖玄烨六十寿辰时所铸
    * 民间传说罗汉钱有辟邪功效，常被用作护身符、定情信物
    * 主要由宝泉局、宝源局铸造
    * 按面文细分有：
      * **大字罗汉**：面文字体较大
      * **小字罗汉**：面文字体较小
      * **长熙罗汉**："熙"字较长
      * **短熙罗汉**："熙"字较短

  * **特殊品类**：
    * **母钱**：铸工精整，字口深峻，用于翻铸流通钱
    * **雕母（祖钱）**：手工雕刻而成，存世极罕
    * **宫钱**：大型厚重，铸工极精，用于宫廷赏赐
    * **罗汉大样**：直径较大（约2.7厘米），铜质厚重，较为少见
    * **西城版**：面文风格特殊，据考为新疆地区所铸
