# 2. 汉代钱币详细介绍

## 2.1 榆荚半两

### 榆荚半两

- **铸造时间**：汉高祖刘邦时期（约公元前206年—前195年）。据《汉书·食货志》记载："汉兴，以为秦钱重难用，更令民铸荚钱。"榆荚半两是西汉建立后最初铸行的钱币。
- **材质成分**：青铜（铜、锡、铅合金），铜含量较低，铅含量偏高。
- **尺寸重量**：
  - 直径：1.0—2.0cm（极轻薄）
  - 厚度：0.05—0.1cm
  - 单枚重量：0.5—2g
- **面背特征**：
  - 形制：圆形方孔，无外廓
  - 面文："半两"二字，文字简化潦草，笔画纤细
  - 钱体极薄，边缘不规整
  - 背面：平素无文
  - 铸工粗糙，钱体厚薄不均
- **核心背景**：
  - 汉初因秦半两过重不便流通，民间私铸轻薄小钱，形如榆树荚果（榆荚），故称"榆荚钱"
  - 榆荚半两导致了严重的通货膨胀，据《汉书》记载："米至石万钱，马一匹则百金"
  - 汉高祖允许民间铸币，导致私铸泛滥，钱币质量急剧下降
  - 榆荚半两是研究西汉初期经济政策的重要实物
  - 吕后执政后多次调整币制，逐步纠正榆荚钱导致的币制混乱
- **版别体系**
  - **极小型榆荚钱**：直径1.0—1.5cm，重量0.5—1g，最为轻薄
  - **小型榆荚钱**：直径1.5—2.0cm，重量1—2g
  - **有文榆荚钱**：面文"半两"二字可辨识
  - **无文榆荚钱**：面文模糊不清或无文字
- **价值参考**
  - 普品：50—150元
  - 美品（面文可辨）：150—300元

## 2.2 八铢半两

### 八铢半两

- **铸造时间**：西汉吕后二年（公元前186年）。据《汉书·高后纪》记载，吕后为纠正榆荚钱导致的币制混乱而铸行八铢半两。
- **材质成分**：青铜（铜、锡、铅合金）。
- **尺寸重量**：
  - 直径：2.5—3.0cm
  - 厚度：0.08—0.15cm
  - 单枚重量：4—5.5g（八铢，约5.2g）
- **面背特征**：
  - 形制：圆形方孔，无外廓
  - 面文："半两"二字，文字较榆荚半两规整
  - 钱体较榆荚半两明显加重加大
  - 背面：平素无文
  - 铸工较榆荚半两改善
- **核心背景**：
  - 吕后为纠正榆荚钱导致的币制混乱而铸行，面文仍为"半两"而实重八铢
  - 八铢半两是西汉初期货币调整的重要尝试，试图恢复钱币的重量标准
  - 但因面文"半两"与实际重量"八铢"不符，仍存在币制混乱
  - 铸行时间不长，后被五分钱取代
  - 存世量较榆荚半两少
- **版别体系**
  - **大型八铢半两**：直径2.8—3.0cm，重量5—5.5g
  - **标准型八铢半两**：直径2.5—2.8cm，重量4—5g
  - **小字型**：面文字体较小
- **价值参考**
  - 普品：200—500元
  - 美品（文字清晰）：500—1,500元

## 2.3 五分钱

### 五分钱

- **铸造时间**：西汉吕后六年（公元前182年）。据《中国钱币大辞典》记载，五分钱是吕后时期又一次货币调整的产物。
- **材质成分**：青铜（铜、锡、铅合金）。
- **尺寸重量**：
  - 直径：2.0—2.3cm
  - 厚度：0.06—0.1cm
  - 单枚重量：2—2.5g
- **面背特征**：
  - 形制：圆形方孔，无外廓
  - 面文："半两"二字，文字较规整
  - 实际重量仅为半两（十二铢）的五分之一，故称"五分钱"
  - 背面：平素无文
- **核心背景**：
  - 五分钱是吕后时期在八铢半两之后又一次货币调整的产物
  - 实重仅为半两的五分之一，面文仍为"半两"，体现了西汉初期面文与实际重量不符的混乱状况
  - 铸行时间不长，至文帝时期被四铢半两取代
  - 存世量较八铢半两少
- **版别体系**
  - **标准型五分钱**：直径2.0—2.3cm
  - **有廓五分钱**：部分铸品可见外廓痕迹
- **价值参考**
  - 普品：200—500元
  - 美品（文字清晰）：500—1,000元

## 2.4 四铢半两

### 四铢半两

- **铸造时间**：西汉文帝五年（公元前175年）。据《汉书·食货志》记载："除盗铸钱令，使民放铸。"文帝改革币制，铸四铢半两，并允许民间铸币。
- **材质成分**：青铜（铜、锡、铅合金）。
- **尺寸重量**：
  - 直径：2.2—2.5cm
  - 厚度：0.08—0.12cm
  - 单枚重量：2.5—3g（四铢，约2.6g）
- **面背特征**：
  - 形制：圆形方孔，部分有外廓
  - 面文："半两"二字，文字较规整
  - 钱面平坦，部分有外廓
  - 背面：平素无文
  - 铸工较前代改善
- **核心背景**：
  - 文帝改革币制，铸四铢半两，面文仍为"半两"而实重四铢
  - 文帝允许民间铸币，导致私铸再度泛滥，但钱币质量较榆荚钱大为改善
  - 著名的"邓通钱"即为文帝赐予宠臣邓通铜山后所铸的四铢半两，铸工精良
  - 吴王刘濞亦在封地铸行四铢半两，铸量巨大
  - 四铢半两铸行时间较长，直至武帝初期仍在沿用
- **版别体系**
  - **有廓四铢半两**：有外廓，铸工较精
  - **无廓四铢半两**：无外廓
  - **大字型**：面文字体较大
  - **小字型**：面文字体较小
  - **邓通钱**：铸工精良，铜质较好
- **价值参考**
  - 普品：100—300元
  - 美品（文字清晰）：300—800元

## 2.5 三铢钱

### 三铢钱

- **铸造时间**：汉武帝建元元年（公元前140年）。据《中国钱币大辞典》记载，三铢钱是武帝即位后第一次货币改革的产物。
- **材质成分**：青铜（铜、锡、铅合金）。
- **尺寸重量**：
  - 直径：2.2—2.4cm
  - 厚度：0.08—0.12cm
  - 单枚重量：2—2.5g（三铢）
- **面背特征**：
  - 形制：圆形方孔，有外廓
  - 面文："三铢"二字，为中国钱币史上首次以实际重量标注面文（此前半两钱面文与实际重量不符）
  - 文字为小篆，笔画规整
  - 背面：平素无文
  - 铸工较精整
- **核心背景**：
  - 三铢钱是中国钱币史上首次以实际重量标注面文的重要铸币
  - 此前的半两钱面文"半两"但实际重量远不足半两，三铢钱首次实现了面文与实际重量的统一
  - 三铢钱行用时间极短（约一年），因与四铢半两并行造成混乱而废止
  - 是武帝货币改革的首次尝试，为后续五铢钱制度的建立奠定了基础
  - 存世量较少，品相完好者较为珍贵
- **版别体系**
  - **标准型三铢钱**：直径2.2—2.4cm，最为常见
  - **大字型**：面文字体较大
  - **有廓型**：外廓清晰
- **价值参考**
  - 普品：1,000—3,000元
  - 美品（文字清晰）：3,000—8,000元

## 2.6 白金三品

### 白金三品

- **铸造时间**：汉武帝元狩四年（公元前119年）。据《汉书·食货志》记载，武帝因财政困难铸行白金三品，为中国历史上最早的官方银铸币。
- **材质成分**：银锡合金（银、锡混合铸造）。
- **尺寸重量**：
  - 龙币（上品）：圆形，重约8两（约125g），值三千
  - 马币（中品）：方形，重约6两（约94g），值五百
  - 龟币（下品）：椭圆形，重约4两（约63g），值三百
- **面背特征**：
  - 龙币：正面铸龙纹，圆形
  - 马币：正面铸马纹，方形
  - 龟币：正面铸龟纹，椭圆形
  - 三品各铸不同纹饰，以纹饰区分面值
  - 背面平素或有简单纹饰
- **核心背景**：
  - 白金三品是中国历史上最早的官方银铸币，开创了银质货币的先河
  - 武帝因对匈奴用兵、财政困难而铸行虚值大钱
  - 白金三品面值虚高，民间私铸严重，不久即废止
  - 铸行时间短（约一年），存世极为稀少
  - 白金三品的失败说明虚值货币难以被民间接受
- **版别体系**
  - **龙币（上品）**：圆形，铸龙纹，值三千
  - **马币（中品）**：方形，铸马纹，值五百
  - **龟币（下品）**：椭圆形，铸龟纹，值三百
- **价值参考**
  - 龙币（美品）：30,000—50,000元
  - 马币（美品）：10,000—30,000元
  - 龟币（美品）：5,000—15,000元

## 2.7 郡国五铢

### 郡国五铢

- **铸造时间**：汉武帝元狩五年（公元前118年）。据《汉书·食货志》记载，武帝废半两钱，改铸五铢钱，令各郡国自行铸造。
- **材质成分**：青铜（铜、锡、铅合金）。
- **尺寸重量**：
  - 直径：2.5—2.6cm
  - 厚度：0.1—0.15cm
  - 单枚重量：3.0—4.0g（标准为五铢，约3.5g）
- **面背特征**：
  - 形制：圆形方孔，有外廓
  - 面文："五铢"二字，"五"字在右，"铢"字在左
  - "五"字交笔缓曲，上下两端略呈竖直
  - "铢"字"金"旁简化为三角形或箭头形
  - 有外廓，部分有好廓（穿孔周围凸起）
  - 背面：有外廓和好廓
- **核心背景**：
  - 郡国五铢是五铢钱制度确立后的第一种五铢钱，由各郡国自行铸造

  - 直径：1.8—2.2cm（剪去外缘后变小）
  - 厚度：0.08—0.12cm
  - 单枚重量：1.5—2.5g（剪去外缘后变轻）
- **面背特征**：
  - 形制：将正常五铢钱的边缘剪下一圈，钱体变小变轻
  - 面文："五铢"，但边缘文字被剪去一部分
  - 外廓被剪去，边缘不规整
  - 背面：外廓被剪去
- **核心背景**：
  - 剪边五铢是东汉晚期至三国时期民间私剪五铢钱的产物
  - 民间将正常五铢钱的外缘剪下，收集铜材私铸小钱，一枚五铢可剪下铜材私铸一枚小钱
  - 剪边五铢的大量出现反映了东汉晚期严重的货币贬值和经济混乱
  - 剪边五铢虽非官方铸行，但在当时流通中大量存在
  - 出土数量极多，是东汉晚期至三国时期墓葬和遗址中常见的钱币
- **版别体系**

  - **重度剪边**：剪去大量外缘，直径1.8—2.0cm
  - **对纹剪边**：剪边后面文仍完整可辨

## 2.17 綖环五铢

### 綖环五铢

- **铸造时间**：东汉晚期至三国时期（约公元2世纪—3世纪）。綖环五铢为民间私剪改制的极端形式。
- **材质成分**：青铜（仅保留五铢钱外缘一圈）。
- **尺寸重量**：
  - 外径：2.0—2.5cm
  - 内径：1.5—2.0cm
  - 厚度：0.08—0.12cm
  - 单枚重量：1—2g
- **面背特征**：
  - 形制：仅保留五铢钱的外缘一圈，内芯被凿去，形成环状
  - 面文：仅保留"五铢"二字的边缘部分，文字残缺
  - 内芯被凿去后另铸小钱（剪边五铢或董卓小钱）
  - 形如细环，故称"綖环"
- **核心背景**：
  - 綖环五铢是东汉晚期货币极度贬值的产物
  - 民间将五铢钱的内芯凿出，内芯铸为小钱，外缘作为綖环钱继续流通
  - 一枚五铢钱被分解为綖环和内芯两枚钱，严重破坏了五铢钱制度
  - 綖环五铢的大量出现标志着东汉货币制度的全面崩溃
  - 出土数量较多，是东汉晚期至三国时期经济混乱的实物证据
- **版别体系**
  - **窄环型**：仅保留极窄的外缘
  - **宽环型**：保留较宽的外缘
- **价值参考**
  - 普品：30—80元
  - 美品（文字可辨）：80—300元

## 2.18 董卓小钱

### 董卓小钱

- **铸造时间**：东汉初平元年（公元190年）前后。据《后汉书·董卓传》记载，董卓入长安后，毁五铢钱、铜人、钟虡铸小钱。
- **材质成分**：青铜（含铜量低，杂质多，铅含量偏高）。
- **尺寸重量**：
  - 直径：1.5—2.0cm
  - 厚度：0.05—0.1cm
  - 单枚重量：1—2g
- **面背特征**：
  - 形制：圆形方孔，极轻薄
  - 面文："五铢"但文字模糊不清，部分几乎无文字可辨
  - 钱体轻薄粗糙，边缘不整
  - 铸造极不精整
  - 背面：平素无文
- **核心背景**：
  - 董卓入长安后，搜刮长安城中铜器（包括五铢钱、铜人、铜台、铜钟）铸行小钱
  - 董卓小钱质量极差，含铜量低，导致严重通货膨胀
  - 据《后汉书》记载："谷一斛至数百万钱"，物价飞涨至天文数字
  - 董卓小钱彻底破坏了两汉以来建立的五铢钱制度
  - 此后三国各政权不得不各自重建货币制度
  - 董卓小钱是东汉货币制度崩溃的标志性产物
- **版别体系**
  - **有文型**：面文"五铢"尚可辨识
  - **无文型**：面文模糊不清，几乎无文字
  - **极小型**：直径1.5cm以下
- **价值参考**
  - 普品：50—100元
  - 美品（面文可辨）：100—500元

| 有文型 | 普品 | 80—150元 | 面文尚可辨识 |
