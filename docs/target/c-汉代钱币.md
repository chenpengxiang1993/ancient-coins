# 2. 汉代钱币详细介绍

## 2.1 榆荚半两

### 榆荚半两

- **铸造时间**：汉高祖刘邦时期（约公元前206年—前195年）。据《汉书·食货志》记载："汉兴，以为秦钱重难用，更令民铸荚钱。"榆荚半两是西汉建立后最初铸行的钱币。
- **材质成分**：青铜（铜、锡、铅合金），铜含量较低（约50%—60%），铅含量偏高（约20%—30%）。汉初民间私铸严重，铜质低劣。出土后表面多呈灰绿色锈蚀，原始铸造色泽偏暗。
- **尺寸重量**：
  - 直径：1.0—2.0cm（极轻薄）
  - 厚度：0.05—0.1cm
  - 单枚重量：0.5—2g
- **面背特征**：
  - 形制：圆形方孔，无外廓
  - 钱体极薄，边缘不规整
  - 铸工粗糙，钱体厚薄不均

  - ***面特征***

    - "半两"二字，文字简化潦草，笔画纤细

  - ***背特征***

    - 平素无文

- **铸造工艺**：采用范铸法（泥范），一范多钱。榆荚半两为西汉初期民间私铸和地方滥铸的轻小半两钱，非官方标准铸币。制作流程为：以粗制泥范浇铸，面文"半两"二字阴文反刻于范上，中央方孔设泥芯；合范浇铸熔融青铜液，因铜量极少，钱体极薄，冷却脱范后简单修整。铸工粗糙，钱体厚薄不均，边缘不规整。据《史记·平准书》记载："汉兴，接秦之弊……米至石万钱，马一匹则百金，于是为秦钱重难用，更令民铸荚钱。"可知榆荚半两为汉初允许民间自铸的轻小货币。
- **核心背景**：
  - 汉初因秦半两过重不便流通，民间私铸轻薄小钱，形如榆树荚果（榆荚），故称"榆荚钱"
  - 榆荚半两导致了严重的通货膨胀，据《汉书》记载："米至石万钱，马一匹则百金"
  - 汉高祖允许民间铸币，导致私铸泛滥，钱币质量急剧下降
  - 榆荚半两是研究西汉初期经济政策的重要实物
  - 吕后执政后多次调整币制，逐步纠正榆荚钱导致的币制混乱
- **版别体系**
  - **极小型榆荚钱**：直径1.0—1.5cm，重量0.5—1g。最为轻薄，钱体几近透明，面文"半两"几乎不可辨识，仅见模糊笔画痕迹。铜质灰暗，铅含量极高
  - **小型榆荚钱**：直径1.5—2.0cm，重量1—2g。钱体轻薄，边缘不规整，面文"半两"可辨识但笔画纤细潦草
  - **有文榆荚钱**：面文"半两"二字可辨识，"半"字笔画简练，"两"字结构尚可分辨。铸工稍好
  - **无文榆荚钱**：面文模糊不清或无文字，钱面仅见铜质凸起痕迹。铸工极为粗率，为私铸泛滥的产物
- **价值参考**
  - 普品：50—150元
  - 美品（面文可辨）：150—300元

## 2.2 八铢半两

### 八铢半两

- **铸造时间**：西汉吕后二年（公元前186年）。据《汉书·高后纪》记载，吕后为纠正榆荚钱导致的币制混乱而铸行八铢半两。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—75%，锡含量约5%—15%，铅含量约5%—20%。出土后表面多呈青绿色锈蚀，原始铸造色泽为金黄色。
- **尺寸重量**：
  - 直径：2.5—3.0cm
  - 厚度：0.08—0.15cm
  - 单枚重量：4—5.5g（八铢，约5.2g）
- **面背特征**：
  - 形制：圆形方孔，无外廓
  - 钱体较榆荚半两明显加重加大
  - 铸工较榆荚半两改善

  - ***面特征***

    - "半两"二字，文字较榆荚半两规整

  - ***背特征***

    - 平素无文

- **铸造工艺**：采用范铸法（泥范），一范多钱。八铢半两为西汉高后吕雉二年（公元前186年）官方铸行的半两钱，是汉初整顿币制的产物。制作流程为：先雕刻八铢半两阴文母范（含面文"半两"二字），中央方孔设泥芯；翻制工作铸范后合范浇铸熔融青铜液，冷却脱范后逐枚修整。八铢半两较榆荚半两明显加重加大，铸工有所改善，但仍无内外廓。
- **核心背景**：
  - 吕后为纠正榆荚钱导致的币制混乱而铸行，面文仍为"半两"而实重八铢
  - 八铢半两是西汉初期货币调整的重要尝试，试图恢复钱币的重量标准
  - 但因面文"半两"与实际重量"八铢"不符，仍存在币制混乱
  - 铸行时间不长，后被五分钱取代
  - 存世量较榆荚半两少
- **版别体系**
  - **大型八铢半两**：直径2.8—3.0cm，重量5—5.5g。钱体厚重，面文"半两"字体端庄，笔画粗壮，字口较深。铸工较精，铜质较好
  - **标准型八铢半两**：直径2.5—2.8cm，重量4—5g。面文"半两"笔画规整，"半"字横画平直，"两"字"人"部较短。铸工尚可
  - **小字型**：面文"半两"字体较小，笔画纤细，字口较浅。布局偏于穿孔一侧。为减重铸行
- **价值参考**
  - 普品：200—500元
  - 美品（文字清晰）：500—1,500元

## 2.3 五分钱

### 五分钱

- **铸造时间**：西汉吕后六年（公元前182年）。据《中国钱币大辞典》记载，五分钱是吕后时期又一次货币调整的产物。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—75%，锡含量约5%—15%，铅含量约5%—20%。出土后表面多呈青绿色锈蚀，原始铸造色泽为金黄色。
- **尺寸重量**：
  - 直径：2.0—2.3cm
  - 厚度：0.06—0.1cm
  - 单枚重量：2—2.5g
- **面背特征**：
  - 形制：圆形方孔，无外廓
  - 实际重量仅为半两（十二铢）的五分之一，故称"五分钱"

  - ***面特征***

    - "半两"二字，文字较规整

  - ***背特征***

    - 平素无文

- **铸造工艺**：采用范铸法（泥范），一范多钱。五分钱为西汉高后吕雉六年（公元前182年）铸行的轻小半两钱。制作流程为：先雕刻五分钱阴文母范（含面文"半两"二字），中央方孔设泥芯；合范浇铸熔融青铜液，冷却脱范后修整。五分钱实际重量仅约为标准半两的五分之一，铸工尚可，但钱体轻薄。
- **核心背景**：
  - 五分钱是吕后时期在八铢半两之后又一次货币调整的产物
  - 实重仅为半两的五分之一，面文仍为"半两"，体现了西汉初期面文与实际重量不符的混乱状况
  - 铸行时间不长，至文帝时期被四铢半两取代
  - 存世量较八铢半两少
- **版别体系**
  - **标准型五分钱**：直径2.0—2.3cm。面文"半两"笔画简练，"半"字横画较短，"两"字结构紧凑。钱体轻薄，无外廓，铸工尚可
  - **有廓五分钱**：部分铸品可见低浅外廓痕迹，外廓不甚规整但可辨识。铸工较标准型精整，较为少见
- **价值参考**
  - 普品：200—500元
  - 美品（文字清晰）：500—1,000元

## 2.4 四铢半两

### 四铢半两

- **铸造时间**：西汉文帝五年（公元前175年）。据《汉书·食货志》记载："除盗铸钱令，使民放铸。"文帝改革币制，铸四铢半两，并允许民间铸币。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—75%，锡含量约5%—15%，铅含量约5%—20%。出土后表面多呈青绿色锈蚀，原始铸造色泽为金黄色。
- **尺寸重量**：
  - 直径：2.2—2.5cm
  - 厚度：0.08—0.12cm
  - 单枚重量：2.5—3g（四铢，约2.6g）
- **面背特征**：
  - 形制：圆形方孔，部分有外廓
  - 钱面平坦，部分有外廓
  - 铸工较前代改善

  - ***面特征***

    - "半两"二字，文字较规整

  - ***背特征***

    - 平素无文

- **铸造工艺**：采用范铸法（泥范或铜范），一范多钱。四铢半两为西汉文帝前元五年（公元前175年）铸行的半两钱，是汉初币制改革的重要成果。制作流程为：先雕刻四铢半两阴文母范（含面文"半两"二字），中央方孔设泥芯，部分钱面有外廓；翻制工作铸范后合范浇铸熔融青铜液，冷却脱范后逐枚修整。四铢半两铸工较前代明显改善，钱面平坦，部分有外廓。据《史记·平准书》记载："至孝文时，荚钱益多，轻，乃更铸四铢钱，其文为'半两'。"
- **核心背景**：
  - 四铢半两是西汉文帝前元五年（公元前175年）铸行的半两钱
  - 文帝允许民间铸币，导致私铸再度泛滥，但钱币质量较榆荚钱大为改善
  - 著名的"邓通钱"即为文帝赐予宠臣邓通铜山后所铸的四铢半两，铸工精良
  - 吴王刘濞亦在封地铸行四铢半两，铸量巨大
  - 四铢半两铸行时间较长，直至武帝初期仍在沿用
- **版别体系**
  - **有廓四铢半两**：有外廓，外廓规整清晰，钱面平坦，铸工较精。面文"半两"笔画端正，字口较深
  - **无廓四铢半两**：无外廓，钱面边缘直截，铸工较有廓版粗率。面文"半两"笔画较为潦草
  - **大字型**：面文"半两"字体较大，笔画粗壮，"半"字横画舒展，"两"字"人"部修长。字口深峻
  - **小字型**：面文字体较小，笔画纤细，字口较浅。布局偏于穿孔上方
  - **邓通钱**：据《史记·佞幸列传》记载，文帝赐宠臣邓通铜山铸钱。铸工精良，铜质纯净细腻，面文规整端庄，边缘修整整齐，为四铢半两中品质最优者
- **价值参考**
  - 普品：100—300元
  - 美品（文字清晰）：300—800元

## 2.5 三铢钱

### 三铢钱

- **铸造时间**：汉武帝建元元年（公元前140年）。据《中国钱币大辞典》记载，三铢钱是武帝即位后第一次货币改革的产物。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—75%，锡含量约5%—15%，铅含量约5%—20%。出土后表面多呈青绿色锈蚀，原始铸造色泽为金黄色。
- **尺寸重量**：
  - 直径：2.2—2.4cm
  - 厚度：0.08—0.12cm
  - 单枚重量：2—2.5g（三铢）
- **面背特征**：
  - 形制：圆形方孔，有外廓
  - 文字为小篆，笔画规整
  - 铸工较精整

  - ***面特征***

    - "三铢"二字，为中国钱币史上首次以实际重量标注面文（此前半两钱面文与实际重量不符）

  - ***背特征***

    - 平素无文

- **铸造工艺**：采用范铸法（铜范或泥范），一范多钱。三铢钱为西汉武帝建元元年（公元前140年）铸行，是中国钱币史上首次以实际重量标注面文的货币。制作流程为：先雕刻三铢钱阴文母范（含面文"三铢"二字，小篆书体），有外廓，中央方孔设泥芯或金属芯；翻制工作铸范后合范浇铸熔融青铜液，冷却脱范后逐枚修整。三铢钱铸工较精整，有外廓，形制规整。据《汉书·食货志》记载，武帝"罢半两钱，铸三铢钱"，但因重量过轻，流通时间仅数年即被废止。
- **核心背景**：
  - 三铢钱是中国钱币史上首次以实际重量标注面文的重要铸币
  - 此前的半两钱面文"半两"但实际重量远不足半两，三铢钱首次实现了面文与实际重量的统一
  - 三铢钱行用时间极短（约一年），因与四铢半两并行造成混乱而废止
  - 是武帝货币改革的首次尝试，为后续五铢钱制度的建立奠定了基础
  - 存世量较少，品相完好者较为珍贵
- **版别体系**
  - **标准型三铢钱**：直径2.2—2.4cm。面文"三铢"二字小篆规整，"三"字三横平行等距，"铢"字"金"旁简化为三角形。有外廓，铸工较精整。最为常见
  - **大字型**：面文"三铢"字体较大，笔画粗壮，"三"字横画舒展，"铢"字结构宽博。字口深峻。较为罕见
  - **有廓型**：外廓高而规整，好廓（穿孔周围廓线）清晰可辨，钱面平坦。为向五铢钱过渡的标准形制，较少见
- **价值参考**
  - 普品：1,000—3,000元
  - 美品（文字清晰）：3,000—8,000元

## 2.6 白金三品

### 白金三品

- **铸造时间**：汉武帝元狩四年（公元前119年）。据《汉书·食货志》记载，武帝因财政困难铸行白金三品，为中国历史上最早的官方银铸币。
- **材质成分**：银锡合金，为中国历史上最早的官方银铸币。龙币、马币、龟币三等均以银锡混合铸造。出土后表面呈灰黑色至暗银色。
- **尺寸重量**：
  - 龙币（上品）：圆形，重约8两（约125g），值三千
  - 马币（中品）：方形，重约6两（约94g），值五百
  - 龟币（下品）：椭圆形，重约4两（约63g），值三百
- **面背特征**：
  - 中国历史上最早的官方银铸币，开创银质货币先河
  - 银锡合金铸造，出土后表面呈灰黑色至暗银色
  - 三品分别为不同形制：龙币圆形、马币方形、龟币椭圆形
  - 以纹饰区分面值：龙币值三千、马币值五百、龟币值三百
  - 因币值虚高，铸行不久即废止

  - ***面特征***

    - 龙币（上品）：圆形，正面铸龙纹，龙身蜿蜒盘旋，龙首昂扬，鳞甲纹饰精细
    - 马币（中品）：方形，正面铸马纹，马作行走姿态，四肢健壮，鬃尾飘逸
    - 龟币（下品）：椭圆形，正面铸龟纹，龟甲纹路清晰可辨，四足伸展

  - ***背特征***

    - 三品背面均平素或有简单纹饰

- **铸造工艺**：采用范铸法（精制泥范或金属范），一范一钱。白金三品为西汉武帝元狩四年（公元前119年）发行的银锡合金铸币，是中国历史上最早的政府发行的银质铸币之一。制作流程为：先分别雕刻龙纹（圆形）、马纹（方形）、龟纹（椭圆形）三种阴文铸范，纹饰精细复杂，龙纹、马纹、龟纹均需在范腔中精确雕刻；合范后浇铸银锡合金液（据《汉书·食货志》记载为银锡白金合金），冷却脱范后精心修整打磨。因三种钱币形制各异（圆、方、椭圆），需分别制作不同形状的铸范，工艺复杂度较高。白金三品铸行时间短，因币值虚高，民间盗铸严重，不久即废止。
- **核心背景**：
  - 白金三品是中国历史上最早的官方银铸币，开创了银质货币的先河
  - 武帝因对匈奴用兵、财政困难而铸行虚值大钱
  - 白金三品面值虚高，民间私铸严重，不久即废止
  - 铸行时间短（约一年），存世极为稀少
  - 白金三品的失败说明虚值货币难以被民间接受
- **版别体系**
  - **龙币（上品）**：圆形，正面铸龙纹，龙身蜿蜒盘旋，龙首昂扬，鳞甲纹饰精细。值三千。为三品中最为珍稀者
  - **马币（中品）**：方形，正面铸马纹，马作行走姿态，四肢健壮，鬃尾飘逸。值五百。较少见
  - **龟币（下品）**：椭圆形，正面铸龟纹，龟甲纹路清晰可辨，四足伸展。值三百。为三品中相对多见者
- **价值参考**
  - 龙币（美品）：30,000—50,000元
  - 马币（美品）：10,000—30,000元
  - 龟币（美品）：5,000—15,000元

## 2.7 郡国五铢

### 郡国五铢

- **铸造时间**：汉武帝元狩五年（公元前118年）。据《汉书·食货志》记载，武帝废半两钱，改铸五铢钱，令各郡国自行铸造。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—75%，锡含量约5%—15%，铅含量约5%—20%。出土后表面多呈青绿色锈蚀，原始铸造色泽为金黄色。
- **尺寸重量**：
  - 直径：2.5—2.6cm
  - 厚度：0.1—0.15cm
  - 单枚重量：3.0—4.0g（标准为五铢，约3.5g）
- **面背特征**：
  - 形制：圆形方孔，有外廓
  - "五"字交笔缓曲，上下两端略呈竖直
  - "铢"字"金"旁简化为三角形或箭头形
  - 有外廓，部分有好廓（穿孔周围凸起）

  - ***面特征***

    - "五铢"二字，"五"字在右，"铢"字在左

  - ***背特征***

    - 有外廓和好廓

- **铸造工艺**：采用范铸法（铜范或泥范），一范多钱。郡国五铢为西汉武帝元狩五年（公元前118年）令各郡国铸行的五铢钱，是五铢钱的创始品种。制作流程为：先雕刻五铢钱阴文母范（含面文"五铢"二字），有外廓，部分有好廓，中央方孔设金属芯或泥芯；各郡国自行翻制工作铸范后合范浇铸熔融青铜液，冷却脱范后逐枚修整。因各郡国铸造标准不一，郡国五铢质量参差不齐，轻重厚薄各异。据《汉书·食货志》记载："令诸郡国铸五铢钱，周郭其质，令不可磨取鎔。"
- **核心背景**：
  - 郡国五铢是五铢钱制度确立后的第一种五铢钱，由各郡国自行铸造
  - 五铢钱制度从此确立，沿用至唐武德四年（公元621年），历时七百余年
  - 因各地铸工参差，质量不一，后由中央统一铸造（上林三官五铢）
  - "五铢"面文首次实现了面文与实际重量的完全统一
  - 郡国五铢铸量大，出土数量极多
- **版别体系**
  - **标准型郡国五铢**：直径2.5—2.6cm。面文"五铢"，"五"字交笔缓曲，上下两端略呈竖直；"铢"字"金"旁简化为三角形或箭头形，"朱"部方正。铸工尚可，有外廓。最为常见
  - **粗铸型**：铸工较粗糙，钱体厚薄不均，边缘不甚规整，面文字体粗放潦草。地方郡国铸行，铜质偏灰
  - **精铸型**：铸工较精整，外廓规整高挺，面文"五铢"笔画端正清晰，字口深峻。少数郡国铸行，较为少见
  - **有廓型**：外廓和好廓（穿孔周围凸起廓线）均清晰高挺，钱面平坦规整。较少见
  - **穿下半星**：穿孔下方铸有一凸起圆点（星纹），位于"铢"字下方。较为少见
- **价值参考**
  - 普品：20—50元
  - 美品（铸工精整）：50—200元

## 2.8 赤仄五铢

### 赤仄五铢

- **铸造时间**：汉武帝元鼎二年（公元前115年）。据《汉书·食货志》记载，赤仄五铢由钟官统一铸造，以纠正郡国五铢质量参差的问题。
- **材质成分**：青铜（铜、锡、铅合金），以赤铜（即纯度较高的红铜）为廓，边缘呈赤红色。铜含量较高，质地精良。出土后表面多呈青绿色锈蚀，边缘赤铜部分呈暗红色。
- **尺寸重量**：
  - 直径：2.5—2.6cm
  - 厚度：0.1—0.15cm
  - 单枚重量：3.5—4g
- **面背特征**：
  - 形制：圆形方孔，有外廓和好廓
  - 钱缘以赤铜镶边，外廓高而规整，呈赤红色
  - 一枚赤仄五铢当五枚郡国五铢

  - ***面特征***

    - "五铢"，铸工较郡国五铢精整

  - ***背特征***

    - 有外廓和好廓

- **铸造工艺**：采用范铸法（铜范），一范多钱，工艺较郡国五铢更为精整。赤仄五铢为西汉武帝元鼎二年（公元前115年）中央统一铸行的五铢钱，以赤铜镶边为特征。制作流程为：先以高精度铜范雕刻赤仄五铢阴文母范（含面文"五铢"二字），外廓高而规整，好廓清晰，中央方孔设金属芯；合范浇铸熔融青铜液后，以赤铜镶边处理钱缘，使外廓呈赤红色。铸工较郡国五铢精整，规格统一。据《汉书·食货志》记载："悉禁郡国毋铸钱，专令上林三官铸……民之铸钱益少，计其费不能相当。"
- **核心背景**：
  - 赤仄五铢是西汉武帝元鼎二年（公元前115年）为整顿郡国五铢质量参差而铸行的精整五铢钱
  - 以赤铜为廓是其最大特征，外廓高而规整
  - 因一枚当五枚郡国五铢，虚值严重，民间多有私熔私铸
  - 铸行时间不长（约两年），后由上林三官五铢取代
  - 存世量较郡国五铢少，品相完好者较为珍贵

  - **厚重型**：铜质厚重，钱体饱满，外廓高挺，面文字口深峻清晰。铸工最为精整，较为罕见

## 2.15 中平四出五铢

### 中平四出五铢

- **铸造时间**：东汉灵帝中平三年（公元186年）。据《后汉书·灵帝纪》记载，灵帝铸四出五铢，因背面有四道凸线而得名。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—75%，锡含量约5%—15%，铅含量约5%—20%。出土后表面多呈青绿色锈蚀，原始铸造色泽为金黄色。
- **尺寸重量**：
  - 直径：2.5cm
  - 厚度：0.1—0.15cm
  - 单枚重量：约3g
- **面背特征**：
  - 形制：圆形方孔，有外廓
  - 四出纹是灵帝五铢的典型特征，极具辨识性
  - 铸工尚可

  - ***面特征***

    - "五铢"

  - ***背特征***

    - 穿孔四角各有一道凸线延伸至外廓，故称"四出纹"

- **核心背景**：
  - 四出五铢因其特殊的背纹而易于辨识，是东汉钱币中最具辨识性的品种之一
  - 民间传说"四出"象征天下四方皆出钱，为不祥之兆，预示东汉将亡
  - 黄巾起义爆发于中平元年（公元184年），此后灵帝于中平三年（公元186年）铸行四出五铢，民间传说"四出"为不祥之兆，预示东汉将亡
  - 四出五铢铸量较大，出土数量较多
- **版别体系**
  - **标准型四出五铢**：面文"五铢"笔画端正，背面穿孔四角各有一道凸线延伸至外廓（四出纹），四出纹清晰规整，对称分布。铸工尚可。最为常见
  - **深纹型**：四出纹深峻凸起，线条粗壮，从穿孔四角至外廓一气呵成，极具立体感。铸工较精
  - **浅纹型**：四出纹较浅，线条纤细，需仔细辨认方可辨识。铸工较粗，可能为后期铸行

  - 美品（四出纹清晰）：50—200元

## 2.16 剪边五铢

### 剪边五铢

- **铸造时间**：东汉晚期至三国时期（约公元2世纪—3世纪）。剪边五铢并非官方铸行，而是民间私剪改制。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—75%，锡含量约5%—15%，铅含量约5%—20%。出土后表面多呈青绿色锈蚀，原始铸造色泽为金黄色。
- **尺寸重量**：
  - 直径：1.8—2.2cm（剪去外缘后变小）
  - 厚度：0.08—0.12cm
  - 单枚重量：1.5—2.5g（剪去外缘后变轻）
- **面背特征**：
  - 形制：将正常五铢钱的边缘剪下一圈，钱体变小变轻
  - 外廓被剪去，边缘不规整

  - ***面特征***

    - "五铢"，但边缘文字被剪去一部分

  - ***背特征***

    - 外廓被剪去

- **铸造工艺**：剪边五铢并非独立铸造，而是对正常五铢钱进行剪边加工的产物。制作方式为：将正常流通的五铢钱以外廓为界，用金属工具沿外廓内侧将边缘剪下一圈铜材，收集剪下的铜材用于私铸小钱。剪边后的五铢钱外廓被去除，钱体变小变轻，边缘不规整，面文"五铢"文字部分被剪去。这种剪边行为是东汉晚期至三国时期民间私剪五铢钱的普遍现象，一枚五铢可剪下铜材私铸一枚小钱，反映了严重的货币贬值和经济混乱。
- **核心背景**：
  - 剪边五铢是东汉晚期至三国时期民间私剪五铢钱的产物
  - 民间将正常五铢钱的外缘剪下，收集铜材私铸小钱，一枚五铢可剪下铜材私铸一枚小钱
  - 剪边五铢的大量出现反映了东汉晚期严重的货币贬值和经济混乱
  - 剪边五铢虽非官方铸行，但在当时流通中大量存在
  - 出土数量极多，是东汉晚期至三国时期墓葬和遗址中常见的钱币
- **版别体系**
  - **轻度剪边**：仅剪去少量外缘，直径2.0—2.2cm。外廓被剪去但面文"五铢"完整可辨，"五"字和"铢"字结构清晰。剪切断面较为平整
  - **重度剪边**：剪去大量外缘，直径1.8—2.0cm。面文"五铢"部分文字被剪去，"铢"字"金"旁或"朱"部受损。剪切断面粗糙不整
  - **对纹剪边**：剪边后面文仍完整可辨，"五"字交笔弯曲，"铢"字结构完整。虽外廓被去除，但文字信息保留较好
- **价值参考**
  - 普品：10—30元
  - 美品（面文完整）：30—100元

## 2.17 綖环五铢

### 綖环五铢

- **铸造时间**：东汉晚期至三国时期（约公元2世纪—3世纪）。綖环五铢为民间私剪改制的极端形式。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—75%，锡含量约5%—15%，铅含量约5%—20%。出土后表面多呈青绿色锈蚀，原始铸造色泽为金黄色。
- **尺寸重量**：
  - 外径：2.0—2.5cm
  - 内径：1.5—2.0cm
  - 厚度：0.08—0.12cm
  - 单枚重量：1—2g
- **面背特征**：
  - 形制：仅保留五铢钱的外缘一圈，内芯被凿去，形成环状
  - 内芯被凿去后另铸小钱（剪边五铢或董卓小钱）
  - 形如细环，故称"綖环"

  - ***面特征***

    - 仅保留"五铢"二字的边缘部分，文字残缺

- **铸造工艺**：綖环五铢并非独立铸造，而是对正常五铢钱进行凿芯加工的产物。制作方式为：将正常五铢钱固定后，以金属冲具从钱面中央方孔处冲凿，将内芯凿出，仅保留外缘一圈铜环。凿出的内芯（称为"对文钱"或"剪轮钱"）另作流通使用，外缘铜环即为綖环五铢。一枚五铢钱被分解为綖环和内芯两部分，分别充作两枚钱使用，严重破坏了五铢钱制度。綖环五铢形如细环，面文仅保留"五铢"二字的边缘部分。
- **核心背景**：
  - 綖环五铢是东汉晚期货币极度贬值的产物
  - 民间将五铢钱的内芯凿出，内芯铸为小钱，外缘作为綖环钱继续流通
  - 一枚五铢钱被分解为綖环和内芯两枚钱，严重破坏了五铢钱制度
  - 綖环五铢的大量出现标志着东汉货币制度的全面崩溃
  - 出土数量较多，是东汉晚期至三国时期经济混乱的实物证据
- **版别体系**
  - **窄环型**：仅保留极窄的外缘铜环，环宽约0.1—0.2cm，面文"五铢"仅残留边缘笔画痕迹。铜环纤细如丝
  - **宽环型**：保留较宽的外缘铜环，环宽约0.2—0.3cm，面文"五铢"残留部分较多，"五"字交笔或"铢"字"朱"部尚可辨识
- **价值参考**
  - 普品：30—80元
  - 美品（文字可辨）：80—300元

## 2.18 董卓小钱
