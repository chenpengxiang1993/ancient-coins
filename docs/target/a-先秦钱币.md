# 2. 先秦钱币详细介绍

## 2.1 无文铜贝

### 无文铜贝

- **铸造时间**：商代晚期（约公元前14世纪—前11世纪）至西周（约公元前1046年—前771年），使用延续至春秋初期。据《中国钱币大辞典》记载，安阳殷墟出土的无文铜贝为迄今所见最早的金属铸币实物。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—75%，锡含量约10%—20%，铅含量约5%—15%。据《中国古钱谱》分析，不同产地和时期的铜贝成分存在差异，商代晚期铜含量偏高，西周时期锡含量有所增加。
- **尺寸重量**：
  - 长度：1.5—2.2cm
  - 宽度：1.0—1.5cm
  - 厚度：0.1—0.3cm
  - 单枚重量：2—5g
- **面背特征**：
  - 正面：仿天然海贝凸起形态，中间有一道纵向齿槽，两侧各有一排齿状凸纹（通常8—12齿），纹路沟槽清晰可辨，齿尖圆钝
  - 背面：平面或微凹，打磨较平整，部分保留铸造浇口痕迹
  - 青铜质地特征：表面呈青绿色锈蚀（出土状态），原始色泽为金黄色或浅黄色，质地坚硬
  - 打磨工艺：铸造成型后经手工打磨修整，表面光洁度因品而异，早期打磨粗糙，晚期趋于精细
  - 无文字，故称"无文铜贝"
- **核心背景**：
  - 无文铜贝是中国迄今发现最早的金属铸币，标志着中国货币从天然实物货币向金属铸币的历史性跨越
  - 商代晚期，随着商品交换的发展，天然海贝供不应求，人们开始以青铜仿铸海贝形态的金属货币
  - 出土范围主要集中在河南安阳殷墟、陕西周原、河南洛阳三大商周核心区域
  - 代表性考古发现：安阳大司空村商代墓葬出土无文铜贝百余枚，河南辉县琉璃阁商代墓葬亦有出土
  - 无文铜贝的出现，奠定了中国金属铸币的基本理念，为后世货币发展开创了先河
- **版别体系**
  - **大型铜贝**：体长2.0—2.2cm，重量4—5g，齿纹清晰，铸工较精，商代晚期铸行
  - **中型铜贝**：体长1.7—2.0cm，重量3—4g，数量最多，为最常见的规格
  - **小型铜贝**：体长1.5—1.7cm，重量2—3g，西周至春秋初期铸行
  - **特厚型铜贝**：厚度0.25—0.3cm，重量偏重，铸造较精整，存世较少
- **价值参考**
  - 普品（中型）：800—1,500元
  - 美品（齿纹清晰）：1,500—3,000元

| **版别**      | **品相等级** | **参考价格区间**   | **备注**         |
| ----------- | -------- | ------------ | -------------- |
| 大型铜贝        | 普品       | 1,500—3,000元 | 商代晚期，体长2.0cm以上 |
| 中型铜贝        | 普品       | 800—1,500元   | 最常见规格          |

## 2.2 包金铜贝

### 包金铜贝

- **铸造时间**：西周（约公元前1046年—前771年）至春秋早期（约公元前770年—前650年）。据《中国钱币大辞典》记载，包金铜贝主要出土于西周高等级贵族墓葬。
- **材质成分**：内芯为青铜贝坯（铜、锡、铅合金），外层包裹金箔（金箔含金量约85%—95%）。据《中国古钱谱》记载，金箔厚度极薄，包裹工艺精湛。
- **尺寸重量**：
  - 长度：1.5—2.0cm（略小于无文铜贝平均值）
  - 宽度：1.0—1.4cm
  - 厚度：0.15—0.35cm（含金箔厚度）
  - 金箔厚度：约0.01—0.03mm
  - 单枚重量：3—6g（含金箔）
- **面背特征**：
  - 正面：青铜贝坯外包金箔，金箔紧贴贝面凸起形态，齿槽纹路在金箔上清晰可辨，金箔覆盖范围完整，从贝齿顶端到底缘均匀包裹
  - 背面：金箔包裹至背面边缘，背面平整，金箔与铜贝坯贴合紧密
  - 金箔包裹工艺：先将青铜贝坯打磨修整，再将极薄金箔锤打贴合于铜贝表面，金箔边缘收口整齐
  - 金箔色泽：出土后因氧化呈暗金色至褐色，原始状态为明亮的金黄色
- **核心背景**：
  - 包金铜贝是高等级贵族使用的大额支付货币，其价值远超普通铜贝，兼具货币与礼仪功能
  - 主要出土于西周至春秋早期的高等级贵族墓葬，如河南三门峡虢国墓地、陕西宝鸡茹家庄西周墓
  - 包金铜贝反映了西周时期贵金属（金）与普通金属（铜）并行的货币观念
  - 其金箔包裹工艺体现了西周时期高超的金属加工技术
  - 存世量远少于无文铜贝，考古出土数量极为有限
- **版别体系**
  - **全包裹型**：金箔完整覆盖铜贝坯全部表面，工艺最精，存世最少
  - **半包裹型**：金箔仅覆盖正面及侧面，背面露出铜质，较为常见
  - **厚金箔型**：金箔较厚（0.02—0.03mm），色泽饱满，多为西周早期制品
  - **薄金箔型**：金箔极薄（0.01—0.02mm），部分区域金箔已脱落，多为西周晚期至春秋早期制品
- **价值参考**
  - 普品（金箔部分脱落）：5,000—10,000元
  - 美品（金箔完整）：10,000—20,000元

| **版别**             | **品相等级** | **参考价格区间**     | **备注**                |
| ------------------ | -------- | -------------- | --------------------- |
| 全包裹型               | 美品       | 15,000—30,000元 | 金箔完整覆盖，极为罕见           |
| 半包裹型               | 普品       | 5,000—10,000元  | 金箔部分保留                |

## 2.3 平肩空首布

### 平肩空首布

- **铸造时间**：春秋中晚期（约公元前7世纪—前5世纪）。据《中国钱币大辞典》记载，平肩空首布主要铸行于周王室领地，是空首布中铸行时间较早的品种。
- **材质成分**：青铜（铜、锡、铅合金）。
- **尺寸重量**：
  - 通高：9.5—12.0cm
  - 肩宽：4.5—5.5cm
  - 足宽：5.0—6.0cm
  - 单枚重量：15—30g
- **面背特征**：
  - 形制：整体呈铲形，上端有中空銎部（安装木柄的空腔），銎部呈圆锥形或方锥形，銎口可见泥芯
  - 肩部平直，两足呈弧形（弧足），面身多有三道纵向凸脊
  - 面文：銎部或面身铸有简单文字或符号，多为单字，内容涉及纪数、纪地或吉语，已知面文包括"一""二""三""五""六""七""八""九""十""武""安臧""周""成""吉""尚""金""氏""上""下""大""日""月""井""田""木""工""君""王""府""商""宗""鼎""贞""宝""万""丘""行""门""山""水""火""土""甲""乙""丙""丁""午""巳""未""申""酉""戌""亥""子""丑""寅""卯""辰"
  - 背面：平素无纹或有简单纹饰
  - 铸造工艺：范铸法，一范一钱，文字为阴文反书于范上，铸后呈阳文
- **核心背景**：
  - 平肩空首布是空首布三大类型之一，因肩部平直而得名
  - 主要出土于河南洛阳地区（周王室故地），是周王室铸行的标志性货币
  - 空首布是从实用农具"镈"（铲形农具）向铸币过渡的关键形态，銎部中空是保留农具柄腔的原始特征
  - "布"为"镈"之通假，布币因形似农具"镈"而得名
  - 平肩空首布的面文多为单字，文字古朴简练，是研究先秦文字的重要实物资料
- **版别体系**
  - **大字型**：面文字体较大，笔画粗壮，铸工精整
  - **小字型**：面文字体较小，笔画纤细
  - **无文型**：面身无文字，仅见简单符号或纹饰
  - **纪数型**：面文为数字，包括"一""二""三""五""六""七""八""九""十"
  - **纪地型**：面文为地名或吉语，包括"武""安臧""周""成""吉""尚""金""氏"
- **价值参考**
  - 普品（面文模糊）：3,000—8,000元
  - 美品（面文清晰）：8,000—30,000元

## 2.4 斜肩空首布

### 斜肩空首布

- **铸造时间**：春秋中晚期（约公元前7世纪—前5世纪）。据《中国钱币大辞典》记载，斜肩空首布铸行于周王室领地及晋国核心区域。
- **材质成分**：青铜（铜、锡、铅合金）。
- **尺寸重量**：
  - 通高：8.5—11.0cm
  - 肩宽：4.0—5.0cm
  - 足宽：4.5—5.5cm
  - 单枚重量：12—25g
- **面背特征**：
  - 形制：整体呈铲形，上端有中空銎部，肩部向两侧倾斜（斜肩），双足弧形
  - 面文：铸有文字，常见"武""卢氏""三川"，面文字体较平肩空首布略小
  - 背面：平素无纹
  - 与平肩空首布的主要区别：肩部倾斜而非平直，体型略小
- **核心背景**：
  - 斜肩空首布因肩部倾斜而得名，是空首布三大类型之一
  - 面文"武"字出现频率最高，或与周武王有关，具有纪念意义
  - "卢氏"为地名，位于今河南卢氏一带
  - 主要出土于河南洛阳及晋南地区
  - 斜肩空首布的铸行时间与平肩空首布大致同期，但铸量相对较少
- **版别体系**
  - **"武"字型**：面文"武"字，最为常见

## 2.5 耸肩空首布

  - 晋国是春秋时期最强盛的诸侯国之一，其货币铸造工艺代表了当时的高水平

## 2.6 原始布

### 原始布

- **铸造时间**：西周晚期至春秋早期（约公元前9世纪—前7世纪）。据《中国钱币大辞典》记载，原始布是由实用农具"镈"向金属铸币过渡的中间形态，铸造年代早于空首布。
- **材质成分**：青铜（铜、锡、铅合金），铜含量较高，质地厚重。
- **尺寸重量**：
  - 通高：10.0—16.0cm（体型大且厚重）
  - 肩宽：5.0—8.0cm
  - 足宽：5.5—8.5cm
  - 单枚重量：30—80g
- **面背特征**：
  - 形制：整体呈大型铲形，保留较多实用农具特征，銎部中空但较浅，銎壁厚实
  - 面身宽大厚重，三道纵向凸脊明显
  - 面文：多无文字，少数铸有简单符号
  - 背面：平素无纹
  - 铸造工艺：铸工较粗糙，保留明显的铸造痕迹，体现了由农具向铸币过渡的原始性
- **核心背景**：
  - 原始布是布币体系中最早的形态，是从实用农具"镈"直接演化而来的过渡型铸币
  - 其体大厚重的特征说明尚未完全脱离农具的实用功能，兼具工具与货币双重属性
  - 原始布的发现数量极少，是研究中国金属铸币起源的重要实物证据
  - 据考古资料，原始布主要出土于中原地区（河南、山西一带）
  - 原始布的存世量远少于空首布，品相完好者极为珍贵
- **版别体系**
  - **大型原始布**：通高14.0—16.0cm，銎部较深，最接近实用农具形态
  - **中型原始布**：通高11.0—14.0cm，銎部变浅，向铸币形态过渡
  - **小型原始布**：通高10.0—11.0cm，銎部更浅，已接近空首布形态
- **价值参考**
  - 普品（中型）：5,000—20,000元
  - 美品（完整）：20,000—50,000元

## 2.7 桥足布钱

### 桥足布钱

- **铸造时间**：战国早中期（约公元前5世纪—前4世纪）。据《中国钱币大辞典》记载，桥足布钱主要铸行于魏国，是魏国最具代表性的铸币。
- **材质成分**：青铜（铜、锡、铅合金）。
- **尺寸重量**：
  - 大型：通高6.0—7.0cm，重约10—20g
  - 小型：通高4.0—5.0cm，重约5—10g
- **面背特征**：
  - 形制：平首布的一种，首部扁平，双足之间弧形连接呈桥洞状，故称"桥足布"
  - 面文：常含"釿"字纪重，包括"梁正尚百当寻釿""安邑二釿""安邑一釿""晋阳一釿""梁充釿""梁半尚二百当寻""山阳"

- **核心背景**：
  - 桥足布钱是战国时期魏国的主要流通货币，因双足弧形连接如桥洞而得名
  - 面文中的"釿"字纪重制度反映了魏国完善的货币重量管理体系
  - "安邑"为魏国早期都城（今山西夏县），"梁"为魏国后期都城（今河南开封）
  - 桥足布有大、小两种规格，分别对应"二釿"和"一釿"的面值

  - **"安邑一釿"型**：面文"安邑一釿"，小型桥足布，通高约4.5cm

## 2.18 明刀钱

### 明刀钱

- **铸造时间**：战国时期（约公元前5世纪—前3世纪）。据《中国钱币大辞典》记载，明刀钱是燕国主要流通货币，铸行时间跨度长，铸量巨大。
- **材质成分**：青铜（铜、锡、铅合金）。
- **尺寸重量**：
  - 通长：13.0—14.0cm
  - 刀身宽：1.5—2.0cm
  - 单枚重量：12—19g（弧背明刀14—19g，磬折明刀12—18g，晚期减重至10g左右）
- **面背特征**：
  - 形制：刀身较齐刀窄小，刀身弧度较大，刀柄细长，柄末端有圆环
  - 面文：正面铸一"明"字（或释为"匽""易"字），字形规整或简化，故称"明刀"
  - 背面：多有文字或符号，内容复杂，涉及纪地、纪数
  - 铸工较齐刀粗率，但形制统一
- **核心背景**：
  - 明刀钱是先秦时期出土数量最多的刀币种类，反映燕国发达的商品经济
  - 面文"明"字的释读学界尚有争议，主要有"明""盟""莒""召""泉""匽""易"不下10种解释。学术界普遍倾向于释为"明"，主要是因为该字字形与甲骨文、金文及小篆中的"明"字相近。近来有学者指出此字与"眼"字同源相通，由"眼"转通"燕"，即燕国以国名发行的刀币
  - 明刀在朝鲜半岛、日本列岛亦有出土，是东亚货币文化交流的重要物证
  - 明刀铸量巨大，是先秦钱币收藏中入门级品种
  - 主要出土于河北、北京、辽宁、内蒙古及朝鲜半岛北部
- **版别体系**
  - **弧背明刀**：刀身弧度大，早期铸行，铸工较精
  - **磬折明刀**：刀身在刀身与刀柄连接处有明显折角（磬折），晚期铸行，铸工较粗
  - **直刀型明刀**：刀身较直，介于弧背与磬折之间
  - **背文分类**：背文种类繁多，可分纪数类、纪地类、符号类
- **价值参考**
  - 普品（磬折明刀）：200—500元
  - 美品（弧背明刀）：500—3,000元

## 2.20 齐刀钱

  - 形制：刀身宽大厚重，刀首近三角形，刀柄略带弯曲，柄末端有圆环形刀环
  - 面文：铸于刀身正面，著名者有"齐法化"（三字刀）、"齐建邦长法化"或"齐造邦长法化""齐返邦长法化"（六字刀）、"节墨之法化"（五字刀，即墨刀）、"节墨法化"（四字刀）、"安阳之法化"（五字刀，安阳刀）
  - 近年学术界对面文末二字有新的释读，旧释"法化"意为"法定货币"，新释作"大刀"（据中国国家博物馆资料）

  - 刀环直径与刀首长度的比例为1:7.5，六枚刀币首尾相接可成圆环，符合《周礼·考工记》"合六而成规"的记载
- **核心背景**：
  - 齐刀钱是先秦时期铸工最精整的货币之一，"法化"即"法定货币"之意，体现了齐国完善的货币管理制度
  - 齐国是春秋战国时期东方最强大的诸侯国之一，经济发达，商业繁荣
  - 齐刀面文中的"齐"指国都临淄，"节墨"即即墨（今山东平度东南），"安阳"为齐国城邑（一说今山东莒县附近，一说今山东曹县一带）

  - 三字刀"齐法化"是田齐建国以后通用的法币，通长17.8—19cm，重40.8—52.4g，流通时间最久，铸量最大，是齐刀中最常见的品种

  - **五字刀（安阳之法化）**：面文五字，通长约17—18.5cm，重约40—50g，安阳地方铸造

## 2.22 楚国铸币

### 楚国铸币

- **铸造时间**：春秋晚期至战国晚期（约公元前6世纪—前3世纪）。楚国独立货币体系贯穿整个战国时期，据《中国钱币大辞典》记载，楚国是先秦时期唯一同时使用铜贝、黄金称量货币和布币的国家。
- **材质成分**：青铜（铜贝、布币）、黄金（郢爰金版）。

- **面背特征**：楚国铸币体系包含三大类型，各有不同特征（详见蚁鼻钱、郢爰金版、殊布当釿、连布各条目）。
- **核心背景**：
  - 楚国是先秦时期唯一建立了独立、完整的多元货币体系的国家，同时使用铜贝、黄金称量货币和布币
  - 楚国铜贝（蚁鼻钱）延续商周贝币传统，形似海贝，是南方楚文化的独特货币形态

## 2.29 半两钱

### 半两钱

- **铸造时间**：战国晚期。据《史记·六国年表》记载，秦惠文王二年（公元前336年）"初行钱"，正式确立半两钱为秦国法定货币。此处指战国时期秦国铸行的半两钱，与秦统一后的半两钱有所区别。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约70%，其余为铅、锡。
- **尺寸重量**：
  - 直径：2.5—3.2cm（据《中国大百科全书》记载，秦半两一般直径2.5—3厘米）
  - 厚度：0.15—0.25cm
  - 单枚重量：5—8g（最常见约5g左右，重的可达10g以上）
- **面背特征**：
  - 形制：圆形方孔，无内外廓（早期特征）
  - 面文："半两"二字，大篆书体（战国秦半两用大篆，非小篆），文字古朴雄浑
  - "半两"即半两（十二铢）之意
  - "两"字中的"人"部分较长，俗称"长人两"，为战国秦半两的典型特征
  - 背面：平素无文
  - 铸工较粗率，钱体欠圆，边缘不甚规整，方孔不甚方正，钱肉或厚或薄，体现了战国时期手工铸造的特征
  - 先秦半两的特点：钱体较厚，穿孔较小，文字高挺，无内外廓，铭文用大篆
- **核心背景**：
  - 先秦半两钱是秦国在战国晚期铸行的货币，是秦始皇统一全国货币的直接原型
  - 秦国自商鞅变法后逐步建立圜钱体系，先后铸行"一珠重一两·十二""一珠重一两·十四""半圜"圜钱，至秦惠文王二年（公元前336年）正式确立半两钱为法定货币
  - 半两钱的圆形方孔形制成为此后两千余年中国铜钱的基本形制，据《中国大百科全书》评价，其"奠定了古代中国乃至世界东方钱币体系的基础"
  - 先秦半两与秦统一后的半两在重量和形制上有所区别：先秦半两铸工较粗，钱体欠圆，铭文用大篆；统一后的秦半两更为规范，铭文改用小篆（相传为丞相李斯所书）
  - 先秦半两存世量较少，品相完好者更为珍贵
- **版别体系**
  - **大型厚重半两**：直径3.5—3.8cm，重8—10g以上，铸工较粗，为先秦早期铸行
  - **中型半两**：直径3.2—3.5cm，重6—8g，铸工稍精
  - **小型半两**：直径3.0—3.2cm，重5—6g
  - **大字半两**：面文"半两"二字较大，笔画粗壮
  - **小字半两**：面文字体较小
  - **有廓半两**：部分后期铸品出现外廓，向规范化过渡
- **价值参考**
  - 普品（中型常见型）：2,000—10,000元
  - 美品（文字清晰）：10,000—50,000元
