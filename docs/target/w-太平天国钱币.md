# 2. 太平天国钱币详细介绍

## 2.1 太平天国

### 太平天国

* **铸造时间**：太平天国癸好三年（公元1853年）始铸，至太平天国灭亡（公元1864年）。

* **材质成分**：黄铜（铜锌合金），铜含量约55%—65%，锌含量约25%—35%，锡铅含量约3%—8%。太平天国铸币铜质尚可，铸工较精。黄铜钱呈金黄色，表面多呈深褐色至黑色包浆。部分试铸品为青铜质地。

* **尺寸重量**：

  * 小平直径：约2.5厘米，重约3—4克

  * 折五直径：约3.3厘米

  * 折十直径：约3.8厘米

  * 折五十直径：约4.8厘米

* **面背特征**：

  * 圆形方孔

  * 面文"太平天国"四字，宋体或楷书

  * 背面铸"圣宝"二字（竖读）

  * 铸工精整

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。太平天国钱为太平天国运动（公元1851年—1864年）期间铸行，洪秀全建立太平天国，定都天京（南京）后铸行。制作流程为：先雕刻太平天国阴文母范（含面文"太平天国"四字，宋体或楷书），中央方孔设金属芯；合范浇铸熔融黄铜液，冷却脱范后逐枚修整。铸工精整，以"圣宝"取代传统的"通宝""元宝"，体现其宗教特色。铸于天京、苏州、杭州、徽州。

* **核心背景**：

  * 太平天国运动（公元1851年—1864年）是中国历史上规模最大的农民起义

  * 洪秀全建立太平天国，定都天京（南京），铸行太平天国钱

  * 太平天国钱以"圣宝"取代传统的"通宝""元宝"，体现其宗教特色

  * 据《中国钱币大辞典》记载，太平天国钱铸于天京、苏州、杭州、徽州

* **版别体系**：

  * **按面值分**：
    * **小平版**：直径约2.5厘米，铸量最大
    * **折五版**：直径约3.3厘米，较少见
    * **折十版**：直径约3.8厘米，背"圣宝"
    * **折五十版**：直径约4.8厘米，珍罕

  * **按面文书体分**：
    * **宋体版**：面文为宋体，笔画方正，为最常见的版式
    * **楷书版**：面文为楷书，笔画流畅，较为少见
    * **隐起文版**：面文笔画有隐起现象，较为少见

  * **按背文位置分**：
    * **背"圣宝"竖读版**：背面"圣宝"二字竖读（上下排列），为最常见的版式
    * **背"圣宝"横读版**：背面"圣宝"二字横读（左右排列），较为少见

  * **按铸地分**：据《中国钱币大辞典》记载，太平天国钱主要铸于以下地区：
    * **天京版**（南京）：铸工较精，铸量最大
    * **苏州版**：铸工尚可
    * **杭州版**：铸工尚可
    * **徽州版**：铸量较少

  * **特殊版别**：
    * **大字版**：面文字体较大
    * **小字版**：面文字体较小
    * **阔缘版**：外缘较宽
    * **母钱**：铸工精整，字口深峻

## 2.2 天国通宝

### 天国通宝

* **铸造时间**：太平天国初期（约公元1851年—1853年）。

* **材质成分**：黄铜（铜锌合金），铜含量约55%—65%，锌含量约25%—35%。太平天国早期铸币。黄铜钱呈金黄色，表面多呈深褐色包浆。

* **尺寸重量**：小平直径约2.4厘米，重约3—3.5克。

* **面背特征**：

  * 圆形方孔

  * 面文"天国通宝"四字楷书直读

  * 光背

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。天国通宝为太平天国初期（约公元1851年—1853年）试铸钱币。制作流程为：先雕刻天国通宝阴文母范（含面文"天国通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融黄铜液，冷却脱范后逐枚修整。为太平天国最早的钱币形式之一，后改为"圣宝"制度，铸量极少，存世稀少。

* **核心背景**：

  * 天国通宝为太平天国初期试铸钱币，铸量极少

  * 为太平天国最早的钱币形式之一，后改为"圣宝"制度

  * 存世稀少

  * 据《中国钱币大辞典》记载，天国通宝为太平天国钱币中的珍稀品种

* **版别体系**：
