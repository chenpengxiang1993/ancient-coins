## 2.1 通行泉货

### 通行泉货

* **铸造时间**：辽太祖耶律阿保机时期（约公元907年—916年）。

* **材质成分**：青铜。

* **尺寸重量**：直径约2.4厘米，单枚重量约3.5克。

* **面背特征**：面文"通行泉货"，四字对读，楷书含隶意，背面光素无文。

* **核心背景**：通行泉货为辽最早铸币之一，铸量极少。辽初以物易物为主，钱币使用不广。

* **版别体系**：仅见小平钱。
