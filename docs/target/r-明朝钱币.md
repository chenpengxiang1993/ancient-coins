# 2. 钱币详情

## 2.1 大中通宝

### 基础信息

- **铸造时间**：明太祖朱元璋洪武元年（公元1368年）始铸（此前于元末公元1361年已始铸，此处指明朝建立后的正式铸行）。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—70%，锡含量约5%—10%，铅含量约15%—20%。元末明初铸币，铜质尚可。出土后表面多呈青绿色锈蚀，原始铸造色泽为金黄色。
- **尺寸重量**：

* 小平直径：约2.4厘米，单枚重量约3.5克
* 折二直径：约2.8厘米
* 折三直径：约3.1厘米
* 折五直径：约3.5厘米
* 当十直径：约4.5厘米

### 面背特征

**通用特征**

* 圆形方孔
* 小平钱光背或背纪地
* 铸工较精

**面（正面）特征**

* 面文"大中通宝"四字楷书，对读

**背（反面）特征**

* 背面有纪地（如"京""浙""福""广""鄂""豫""桂""北平"）和纪值（如"十"）多种

### 铸造工艺

采用范铸法（铜范或砂范），一范多钱。大中通宝为朱元璋建立明朝前后铸行。制作流程为：先雕刻大中通宝阴文母范（含面文"大中通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。铸工较精，背面有纪地和纪值多种，纪地纪值制度奠定了洪武通宝的基础。

### 核心背景

* 朱元璋在建立明朝之前即已铸行大中通宝，建立明朝后继续铸行
* 大中通宝的纪地纪值制度奠定了洪武通宝的基础
* 据《中国钱币大辞典》记载，大中通宝是元末明初最重要的过渡性铸币

## 2.13 崇祯通宝

### 基础信息

- **铸造时间**：明思宗朱由检崇祯年间（公元1628年—1644年）。
- **材质成分**：黄铜（铜锌合金）为主，铜含量约55%—65%，锌含量约25%—35%。崇祯末期因财政困难，部分钱币铜质下降，含铅量增加。黄铜钱呈金黄色至暗黄色；劣质钱呈灰黄色。
