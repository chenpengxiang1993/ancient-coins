# 2. 明朝钱币详细介绍

## 2.1 大中通宝

### 大中通宝

* **铸造时间**：明太祖朱元璋洪武元年（公元1368年）始铸（此前于元末公元1361年已始铸，此处指明朝建立后的正式铸行）。

* **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—70%，锡含量约5%—10%，铅含量约15%—20%。元末明初铸币，铜质尚可。出土后表面多呈青绿色锈蚀，原始铸造色泽为金黄色。

* **尺寸重量**：

  * 小平直径：约2.4厘米，单枚重量约3.5克

  * 折二直径：约2.8厘米

  * 折三直径：约3.1厘米

  * 折五直径：约3.5厘米

  * 当十直径：约4.5厘米

* **面背特征**：

  * 圆形方孔

  * 面文"大中通宝"四字楷书，对读

  * 背面有纪地（如"京""浙""福""广""鄂""豫""桂""北平"）和纪值（如"十"）多种

  * 小平钱光背或背纪地

  * 铸工较精

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。大中通宝为朱元璋建立明朝前后铸行。制作流程为：先雕刻大中通宝阴文母范（含面文"大中通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。铸工较精，背面有纪地和纪值多种，纪地纪值制度奠定了洪武通宝的基础。

* **核心背景**：

  * 朱元璋在建立明朝之前即已铸行大中通宝，建立明朝后继续铸行

  * 大中通宝的纪地纪值制度奠定了洪武通宝的基础

  * 据《中国钱币大辞典》记载，大中通宝是元末明初最重要的过渡性铸币

* **版别体系**：

  * **小平光背版**：背面无文

  * **小平背纪地版**：背铸"京""浙""福""广""鄂""豫""桂""北平"纪地文字

  * **折二版**：背纪地纪值

  * **折三版**：背纪地纪值

  * **折五版**：背纪地纪值

  * **当十版**：背纪地纪值，直径约4.5厘米

## 2.2 洪武通宝

### 洪武通宝

* **铸造时间**：明太祖朱元璋洪武年间（公元1368年—1398年）。

* **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—70%，锡含量约5%—10%，铅含量约15%—20%。明初铸币铜质精良，标准化程度高。出土后表面多呈青绿色锈蚀，原始铸造色泽为金黄色。

* **尺寸重量**：

  * 小平钱直径：2.4—2.5厘米，单枚重3.5—4克

  * 折二钱直径：约2.8厘米

  * 折三钱直径：约3.1厘米

  * 折五钱直径：约3.8厘米

  * 折十钱直径：4.4—4.5厘米，单枚重20—25克

* **面背特征**：

  * 圆形方孔，内外郭规整

  * 面文"洪武通宝"四字楷书，对读，笔画端庄有力

  * 背面纪地、纪值

  * 分为小平、折二、折三、折五、折十五等制

  * 纪地版共九种铸局：京（应天府，今南京）、北平（北平行省）、鄂（湖北省）、浙（浙江省）、济（山东省）、桂（广西省）、福（福建省）、豫（河南省）、广（广东省），各有小平、折二、当三、当五、当十五等

  * 青铜质，铸工精良，标准化程度高

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。洪武通宝为明太祖朱元璋洪武年间（公元1368年—1398年）铸行，为明代开国钱币。制作流程为：先雕刻洪武通宝阴文母范（含面文"洪武通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。铜质精良，铸工精良，标准化程度高。设铸局于各省，实行严格的纪地纪值制度，共九局（京、北平、鄂、浙、济、桂、福、豫、广），各有五等面值。

* **核心背景**：

  * 明代开国钱币，确立了明代制钱的基本形制与五等钱制

  * 是明代铸行时间最长、版别最丰富的钱币之一

  * 奠定了明代三百年的铸币制度基础

  * 据《中国钱币大辞典》记载，洪武通宝设铸局于各省，实行严格的纪地纪值制度

* **版别体系**：

  * **小平光背版**：背面无文

  * **小平背纪重版**：背铸"一钱"等纪重文字

  * **折二背纪重版**：背铸"二钱"

  * **折三背纪重版**：背铸"三钱"

  * **折五背纪重版**：背铸"五钱"

  * **折十背纪重版**：背铸"十两"或纪地

  * **纪地版系列**：背铸各省铸局名称，共九局（京、北平、鄂、浙、济、桂、福、豫、广），各有五等面值。其中背桂小平钱有穿上桂和穿上下"桂一"两种。据《武陵藏珍》记载，洪武通宝钱有四类六十一种，纪地版实共四十六种

## 2.3 洪熙通宝

* **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—70%，锡含量约5%—10%，铅含量约15%—20%。洪熙年间明朝尚未采用黄铜铸币，仍以传统青铜铸造。出土后表面多呈青绿色锈蚀，原始铸造色泽为金黄色。

* **尺寸重量**：小平制式，具体尺寸重量数据因存世极罕而缺少系统记载。

* **面背特征**：

  * 圆形方孔

  * 面文"洪熙通宝"四字楷书直读

  * 光背无文

  * 铸工精整

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。洪熙通宝为明仁宗朱高炽洪熙元年（公元1425年）铸行。制作流程为：先雕刻洪熙通宝阴文母范（含面文"洪熙通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。铸工精整。洪熙年号使用极短，在位仅约十个月即驾崩，铸币量极少，存世极罕。

* **核心背景**：

* **版别体系**：仅见小平钱。

## 2.4 永乐通宝

### 永乐通宝

* **铸造时间**：明成祖朱棣永乐六年（公元1408年）始铸，铸行至永乐末年（公元1424年）。

* **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—70%，锡含量约5%—10%，铅含量约15%—20%。永乐年间仍以传统青铜铸造，铜质精整。出土后表面多呈青绿色锈蚀，原始铸造色泽为金黄色。

* **尺寸重量**：直径2.4—2.5厘米，厚度约0.1厘米，单枚重3.5—4克。

* **面背特征**：

  * 圆形方孔，内外郭规整

  * 面文"永乐通宝"四字楷书，对读，笔画端庄秀丽

  * 钱文深峻

  * 背面平夷无文

  * 青铜质，铜质精良，铸工极为规整，标准化程度极高

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。永乐通宝为明成祖朱棣永乐六年（公元1408年）始铸。制作流程为：先雕刻永乐通宝阴文母范（含面文"永乐通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。铜质精良，铸工极为规整，标准化程度极高，是明代铸币的代表作。铸量极大，是郑和下西洋的主要流通货币。

* **核心背景**：

  * 明代官方法定货币，主要用于海外贸易

  * 据《中国钱币大辞典》记载，永乐通宝铸工精整，是明代铸币的代表作

  * **大字版**：钱文字体较大

  * **细字版**：钱文字体较细

## 2.5 宣德通宝

* **面背特征**：

  * 圆形方孔，宽缘窄郭，缘郭深峻

* **面背特征**：

  * 圆形方孔

  * 面文"弘治通宝"四字楷书直读

  * 光背无文

  * 青铜质

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。弘治通宝为明孝宗弘治年间铸行。制作流程为：先雕刻弘治通宝阴文母范（含面文"弘治通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。弘治朝大力整顿币制，铸造较为规整，但因铜源紧张，部分钱币质量不佳。

* **核心背景**：

    * **光背版**：背面无文，为较常见的版式
    * **背星月版**：背面铸星纹或月纹，有背上星、背下星、穿左星、穿右星、背月纹等多种位置
    * **背纪局版**：背面铸纪局文字，如"工""户""兵""监""院""局"等
    * **背纪重版**：背面铸纪重文字，如"一分""八厘""一钱""五厘"等
    * **背天干地支版**：背面铸天干（如"甲""乙""丙""丁"）或地支文字
    * **背纪值版**：背面铸纪值文字
    * **跑马崇祯**：背面铸奔马图案，因民间附会"一马乱天下"的谶语而著名，为崇祯通宝中的名誉品
    * **背奔马版**：与跑马崇祯同类，马形姿态各异

  * **按面文风格分**：
    * **大字版**：面文字体较大
    * **小字版**：面文字体较小
    * **阔缘版**：外缘较宽
    * **窄缘版**：外缘较窄
    * **粗字版**：笔画粗壮
    * **细字版**：笔画纤细
    * **阔"崇"版**："崇"字较宽
    * **狭"崇"版**："崇"字较窄
    * **长"祯"版**："祯"字较长
    * **短"祯"版**："祯"字较短

  * **按铸地分**：明末各地纷纷铸钱，据《中国钱币大辞典》记载，主要铸地有南京（南铸）、北京（北铸），各省亦有铸行，不同铸地的钱币在文字风格、铜质、铸工等方面均有差异

  * **折二版**：折二钱较少见，直径约2.8厘米
