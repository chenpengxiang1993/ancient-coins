# 2. 钱币详情

## 2.1 秦半两钱

### 基础信息

- **铸造时间**：秦始皇统一六国后（公元前221年）至秦亡（公元前206年）。据《史记·平准书》记载："及至秦，中一国之币为三等，黄金以溢名为上币；铜钱识曰半两，重如其文，为下币。"秦半两钱是以战国秦半两为基础，统一全国货币后在全国范围内推行的标准铸币。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约65%—75%，锡含量约10%—15%，铅含量约5%—15%。出土后表面多呈青绿色或黑漆古锈蚀，原始铸造色泽为深金黄色。
- **尺寸重量**：

- 直径：3.0—3.7cm（多数在3.2—3.5cm之间）
- 穿孔：0.8—1.2cm（方形）
- 厚度：0.15—0.3cm
- 单枚重量：6—10g（标准为十二铢，约8g；实际多有偏差）

### 面背特征

**通用特征**

- 形制：圆形方孔，钱面平坦，无内外廓
- 文字书体：小篆（秦统一文字后的标准书体），相传为丞相李斯所书，笔画粗犷有力，字体结构端庄
- "半两"二字凸起于钱面，字体较高，笔画深峻
- 铸造工艺：范铸法（泥范或石范），铸口痕迹多在钱体上方，铸造精度因品而异
- 与先秦半两的主要区别：先秦半两铸工较粗，钱体欠圆，铭文用大篆；秦统一后的半两更为规范，铭文改用小篆，"两"字中的"人"部分较先秦短

**面（正面）特征**

- "半两"二字，左右排列于穿孔两侧，"半"字在右，"两"字在左（偶有反置）

**背（反面）特征**

- 平素无文无纹，铸面较平

### 铸造工艺

采用范铸法（泥范或石范），一范多钱。据《中国钱币大辞典》记载，秦始皇统一六国后在全国推行统一的半两钱铸造标准。制作流程为：先雕刻半两钱阴文母范（含面文"半两"二字，小篆书体，阴文反刻），中央方孔处设置金属芯或泥芯；翻制工作铸范后合范浇铸熔融青铜液（铜含量约60%—70%），铸口痕迹多在钱体上方；冷却脱范后逐枚修整打磨。秦统一后的半两较先秦半两更为规范，铭文改用小篆，钱体较圆整，但铸造精度因品而异，仍保留一定的手工铸造差异。据《史记·平准书》记载："铜钱识曰半两，重如其文。"

### 核心背景

- 秦半两钱是中国货币史上具有划时代意义的铸币，秦始皇统一六国后废除六国旧币（布币、刀币、圜钱、蚁鼻钱），以秦制"半两"铜钱为全国统一货币
- 据《中国大百科全书》评价，半两钱"奠定了古代中国乃至世界东方钱币体系的基础"
- "半两"意为半两（十二铢），是中国货币史上首次以重量为面文的全国统一铸币
- 秦半两确立了"圆形方孔"的中国铜钱基本形制，圆形方孔的设计寓意"天圆地方"，兼具实用性与文化象征意义
- 这一形制此后沿用两千余年，直至清末宣统通宝为止
- 秦半两的统一铸行，标志着中国货币从多元体系（布币、刀币、圜钱、蚁鼻钱并行）向统一形制的历史性转变
