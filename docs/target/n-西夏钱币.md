采用范铸法（铜范或砂范），一范多钱。藩国宝钱为西夏时期特殊铸币，具体年代不详。制作流程为：先雕刻藩国宝钱阴文母范（含面文"藩国宝钱"西夏文），中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。西夏文钱文，存世极罕。

### 核心背景

藩国宝钱为西夏特殊铸币，存世极罕。马定祥十级制定为三级（罕贵）。
