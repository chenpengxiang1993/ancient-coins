# 2. 西夏钱币详细介绍

## 2.1 福圣宝钱

### 福圣宝钱

* **铸造时间**：西夏毅宗李谅祚福圣承道年间（公元1053年—1056年）。

* **材质成分**：青铜。

* **尺寸重量**：直径约2.4厘米，单枚重量约3.5克。

* **面背特征**：面文"福圣宝钱"为西夏文（又称蕃书），由西夏文字创制者野利仁荣创制。背面光素无文。

* **核心背景**：福圣宝钱为西夏最早的铸币之一，使用西夏文（蕃书）钱文。马定祥十级制定为三级（罕贵）。

* **版别体系**：仅见小平钱。

## 2.7 天盛元宝

### 天盛元宝

* **铸造时间**：西夏仁宗李仁孝天盛年间（公元1149年—1169年）。

* **材质成分**：青铜。

* **尺寸重量**：

  * 直径：约2.4厘米

  * 单枚重量：约3.5克

* **面背特征**：

  * 面文"天盛元宝"，汉文，四字旋读，楷书

  * 铸工较精

  * 背面光素无文

* **核心背景**：

  * 天盛元宝为西夏铸量最大的钱币，也是西夏钱币中最为常见的品种

  * 天盛年间为西夏鼎盛时期，仁宗推行文治，铸钱规模较大

  * 马定祥十级制定为七级（甚少）

* **版别体系**：

  * **光背版**：背面无文

  * **铁母版**：铁钱的母钱，铜质，极少见
