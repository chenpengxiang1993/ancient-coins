# 2. 南朝钱币详细介绍

## 2.1 元嘉四铢

### 元嘉四铢

- **铸造时间**：南朝宋元嘉二十四年（公元447年）。据《宋书·食货志》记载，宋文帝铸行四铢钱，结束了南朝长期不铸新钱的局面。
- **材质成分**：青铜（铜、锡、铅合金）。
- **尺寸重量**：
  - 直径：约2.2—2.4cm
  - 厚度：0.08—0.12cm
  - 单枚重量：约2—3g
- **面背特征**：
  - 形制：圆形方孔，有外廓
  - 面文："四铢"，二字对读
  - 铸工较精
  - 背面：平素无文
- **核心背景**：
  - 元嘉四铢是南朝宋恢复铸币后铸行的第一种钱币
  - 自东晋以来，南方长期沿用旧五铢钱，未大规模铸行新钱
  - 宋文帝元嘉年间政治清明，经济繁荣，恢复铸行四铢钱
  - 元嘉四铢结束了南朝长期不铸新钱的局面
- **版别体系**
  - **标准型元嘉四铢**：直径2.2—2.4cm
- **价值参考**
  - 普品：100—300元
  - 美品（文字清晰）：300—1,000元

## 2.3 孝建四铢

### 孝建四铢

- **铸造时间**：南朝宋孝建元年（公元454年）。据《宋书·食货志》记载，宋孝武帝改元"孝建"后铸行孝建四铢。
- **材质成分**：青铜（铜、锡、铅合金）。
- **尺寸重量**：
  - 直径：约2.2—2.4cm
  - 厚度：0.08—0.12cm
  - 单枚重量：约2—3g
- **面背特征**：
  - 形制：圆形方孔，有外廓
  - 面文："孝建"，二字对读，以年号为面文
  - 文字为薤叶篆（笔画纤细如薤叶），秀美规整
  - 背面：部分铸有"四铢"二字，部分光背
  - 铸工较精
- **核心背景**：
  - 孝建四铢以南朝宋年号"孝建"为面文，是南朝钱币中以年号为面文的代表
  - 薤叶篆书体纤细秀美，是南朝钱币书法的特色
  - 孝建四铢铸量较大，出土数量在南朝钱币中较多
  - 背"四铢"者为初铸版，光背者为后期减重版
- **版别体系**
  - **背"四铢"版**：背面铸有"四铢"二字，初铸版
  - **光背版**：背面平素无文，后期版
  - **大字版**：面文字体较大
  - **小字版**：面文字体较小
- **价值参考**
  - 普品（光背）：50—150元
  - 美品（背"四铢"）：200—800元
