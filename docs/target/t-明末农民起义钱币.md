# 2. 明末农民起义钱币详细介绍

## 2.1 大顺通宝

### 大顺通宝

* **铸造时间**：明末农民起义时期（公元1644年），李自成所铸。

* **材质成分**：青铜。

* **尺寸重量**：小平直径约2.5厘米，重约3—3.5克。

* **面背特征**：

  * 圆形方孔

  * 面文"大顺通宝"四字楷书直读

  * 光背为主

  * 铸工尚精

* **核心背景**：

  * 明末农民起义领袖李自成于公元1644年攻入北京，建立大顺政权，铸行大顺通宝

  * 李自成在北京仅四十余日即败退，大顺政权迅速覆灭

  * 大顺通宝铸期极短，铸量有限

  * 据《中国钱币大辞典》记载，大顺通宝为明末起义军铸币中的稀少品种

* **版别体系**：

  * **小平光背版**：背面无文

  * **小平背星版**：背面铸星纹

## 2.2 西王赏功

### 西王赏功

* **铸造时间**：明末农民起义时期（公元1644年前后），张献忠所铸。

* **材质成分**：金、银、铜三种材质。

* **尺寸重量**：据记载直径约5.0厘米，属大钱型制，具体重量因存世极罕而缺少系统记载。

* **面背特征**：

  * 圆形方孔

  * 面文"西王赏功"四字楷书直读

  * 光背无文

  * 铸工精整

* **核心背景**：

  * 张献忠在四川建立大西政权，自称"西王"，铸"西王赏功"钱

  * 此钱为赏功钱性质，非流通货币，用于赏赐有功将士

  * 分金、银、铜三等，对应不同功勋等级

  * 存世极罕，属中国古泉五十名珍之一

  * 2016年四川彭山江口沉银遗址出水了部分西王赏功钱币，引起了广泛关注

  * 据《中国钱币大辞典》记载，西王赏功钱为明末钱币中的顶级珍品

* **版别体系**：

  * **金质版**：黄金铸造，最高等级赏功钱，存世仅数枚

  * **银质版**：白银铸造，中等级赏功钱

  * **铜质版**：青铜铸造，基础等级赏功钱

## 2.3 永昌通宝

### 永昌通宝

* **铸造时间**：明末农民起义时期（公元1644年—1645年），李自成所铸。

* **材质成分**：青铜。

* **尺寸重量**：

  * 小平直径：约2.5厘米，重约3—3.5克

  * 折五直径：约3.5厘米

  * 折十直径：约4.5厘米

* **面背特征**：

  * 圆形方孔

  * 面文"永昌通宝"四字楷书直读

  * 有小平、折五、折十多种面值

  * 铸于西安、北京

  * 铸工尚精

* **核心背景**：

  * 李自成于公元1644年在西安称帝，国号"大顺"，建元"永昌"，铸行永昌通宝

  * 永昌通宝铸量较大，是明末起义军铸币中最常见的品种

  * 据《中国钱币大辞典》记载，永昌通宝小平钱存世量相对较多

* **版别体系**：

  * **小平版**：楷书直读，光背

  * **折五版**：较少见

  * **折十版**：较少见

## 2.4 兴朝通宝

### 兴朝通宝

* **铸造时间**：明末（公元1647年前后），孙可望所铸。

* **材质成分**：青铜。

* **尺寸重量**：

  * 小平直径：约2.5厘米，重约3—3.5克

  * 折五直径：约3.5厘米

  * 折十直径：约4.5厘米

* **面背特征**：

  * 圆形方孔

  * 面文"兴朝通宝"四字楷书直读

  * 背面铸有纪值文字（如"五厘""一分"）

  * 铸于云南

* **核心背景**：

  * 孙可望为张献忠大西军部将，张献忠死后率部入云南

  * 孙可望在云南铸行兴朝通宝，以稳定地方经济

  * 兴朝通宝铸量较大，流通于云南一带

  * 据《中国钱币大辞典》记载，兴朝通宝为明末清初云南地区的重要铸币

* **版别体系**：

  * **小平版**：背"五厘"

  * **折五版**：背"一分"

  * **折十版**：背"一分"
