# 2. 唐朝钱币详细介绍

## 2.1 开元通宝

### 开元通宝

- **铸造时间**：唐高祖武德四年（公元621年）始铸，贯穿整个唐代，并延续至五代十国及宋初。据《中国钱币大辞典》记载，开元通宝是中国历史上铸行时间最长的钱币之一，前后流通近三百年。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—75%，锡含量约5%—10%，铅含量约10%—20%。唐代早期部分开元通宝含白铜（铜镍合金），呈银白色；晚期因铜材匮乏出现铁钱、铅钱。另有特殊材质如鎏金钱、银钱等。青铜钱出土后表面多呈青绿色或黑漆古锈蚀，原始铸造色泽为金黄色。
- **尺寸重量**：
  - 直径：2.4—2.5厘米
  - 厚度：0.1—0.15厘米
  - 单枚重量：3.5—4.0克（标准重量为一钱，约3.73克）
- **面背特征**：
  - 面文"开元通宝"，四字对读（上下左右）或旋读
  - 文字由书法家欧阳询书写，楷书含隶意，端庄雄浑
  - "开元"并非年号，而是"开创新纪元"之意
  - 外廓规整，好廓清晰
  - 背面：早期光背无文，后期出现星月纹、地名
- **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。开元通宝始铸于唐高祖武德四年（公元621年），面文由书法家欧阳询书写，楷书含隶意。制作流程为：先雕刻开元通宝阴文母范（含面文"开元通宝"四字），有外廓和好廓，中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。唐代早期铸工精整，规格统一，标准重量为一钱（约3.73克）。会昌年间（公元845年）毁佛铸钱，各州设钱监铸行，背面铸州名以示铸地，共发现背字二十三种。晚期铸工渐粗，出现减重现象。开元通宝是中国历史上首次采用"通宝"钱制的钱币，终结了沿用七百余年的五铢钱体制。
- **核心背景**：
  - 开元通宝是中国货币史上划时代的里程碑，终结了自汉武帝以来沿行七百余年的"五铢"钱体制
  - 开创了以"通宝""元宝"为名的钱币命名制度（通宝钱制），影响直至清末
  - "钱"作为重量单位（一两的十分之一）即源于开元通宝的重量标准
  - 铸行近三百年，版别极其复杂，是中国古钱币中版别最多的品种之一
- **版别体系**：
  - **大字**："开"字宽阔，"元"字较大
  - **遒劲**：笔画苍劲有力，字口深峻，铸造极精
  - **容弱**：字体柔弱、笔画纤细
  - **短头元**："元"字第一笔短而平直
  - **会昌开元**：唐武宗会昌五年（公元845年）毁佛铸钱，背面铸有州名或地名，共发现背字二十三种（昌、京、洛、益、蓝、襄、荆、越、宣、洪、潭、兖、润、鄂、平、兴、梁、广、梓、福、丹、桂、永）
  - **星月纹系列**：背面铸有星、月纹饰，月纹按形态和位置分为仰月、附月、侧月、背双月、背上星、背下星、背三月、背四月多种
  - **特殊纹样**：背祥云、背瑞雀、背同心结
  - **特殊材质**：鎏金钱、银钱、白铜钱
- **价值参考**：
  - 普通光背：普品，5—15元
  - 背星月：美品，100—300元

## 2.2 乾封泉宝

### 乾封泉宝

- **铸造时间**：唐高宗乾封元年（公元666年）始铸，铸行仅一年即废止。据《中国钱币大辞典》记载，乾封泉宝是唐代第一枚年号钱，因民间不便而迅速停铸。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—75%，锡含量约5%—10%，铅含量约10%—20%。唐代铜质总体较好，早期部分含白铜（铜镍合金），晚期因铜材匮乏出现铁钱、铅钱。青铜钱出土后表面多呈青绿色或黑漆古锈蚀，原始铸造色泽为金黄色；白铜钱呈银白色；铁钱呈黑褐色。
- **尺寸重量**：
  - 直径：2.5—2.7厘米
  - 厚度：0.1—0.15厘米
  - 单枚重量：3.5—5.0克
- **面背特征**：
  - 面文"乾封泉宝"，四字旋读
  - 楷书字体，笔画端庄
  - "乾封"为唐高宗年号
  - 外廓较宽，好廓规整
  - 背面：光背无文
- **铸造工艺**：采用范铸法（铜范），一范多钱。乾封泉宝为唐高宗乾封元年（公元666年）铸行，是唐代第一枚年号钱，以一当十枚开元通宝。制作流程为：先雕刻乾封泉宝阴文母范（含面文"乾封泉宝"四字，楷书），有外廓和好廓，中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。铸工尚精，但因虚币作价过高，民间流通不便，商贾罢市，铸行仅一年即被废止。
- **核心背景**：
  - 乾封泉宝是唐代第一枚以年号命名的钱币，也是中国历史上较早的年号钱之一
  - 唐高宗封禅泰山后改元乾封，同时铸行新钱，以一当十枚开元通宝
  - 因虚币作价过高，民间流通不便，商贾罢市，铸行仅一年即被废止，恢复开元通宝
  - 铸行时间极短，铸量有限，存世较少，为唐代钱币中的名誉品
- **版别体系**：
  - **大字版**：面文字体较大，笔画粗壮，铸工较精
  - **小字版**：面文字体较小，笔画纤细
  - **狭穿版**：穿孔较窄，文字布局紧凑
  - **阔缘版**：外廓较宽，边缘厚实

## 2.3 乾元重宝

### 乾元重宝

- **铸造时间**：唐肃宗乾元元年（公元758年）始铸。据《中国钱币大辞典》记载，安史之乱期间，朝廷为筹措军费，由御史中丞第五琦建议铸行乾元重宝。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约55%—65%，锡含量约5%—10%，铅含量约15%—25%。晚期因铜材匮乏，四川地区出现铁钱、铅钱。青铜钱出土后表面多呈青绿色锈蚀；铁钱呈黑褐色至红褐色。
- **尺寸重量**：
  - 小平钱直径：2.4—2.6厘米，单枚重量：3.5—4.5克
  - 折二直径：2.7—2.9厘米，单枚重量：5.0—7.0克
  - 折三直径：3.0—3.3厘米，单枚重量：7.0—10.0克
  - 折五直径：3.3—3.6厘米，单枚重量：9.0—12.0克
  - 折十直径：3.6—4.2厘米，单枚重量：10.0—15.0克
- **面背特征**：
  - 面文"乾元重宝"，四字对读
  - 楷书字体，笔画端庄
  - "乾元"为唐肃宗年号
  - 外廓规整，好廓清晰
  - 背面：光背为主，部分有星月纹、祥云纹
- **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。乾元重宝为唐肃宗乾元元年（公元758年）始铸，安史之乱期间为筹措军费由御史中丞第五琦建议铸行，以一当十枚开元通宝。制作流程为：先雕刻乾元重宝阴文母范（含面文"乾元重宝"四字，楷书），有外廓和好廓，中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。铸行初期铸工尚精，后期因民间私铸盛行、减重严重，铸工渐粗。有小平、折二、折三、折五、折十多种规格，晚期四川地区出现铁钱、铅钱。
- **核心背景**：
  - 安史之乱爆发后，唐朝财政极度困难，唐肃宗采纳第五琦建议铸行大钱
  - 乾元重宝初铸以一当十枚开元通宝，与开元通宝并行流通
  - 次年又铸重轮乾元重宝，以一当五十，引发严重通货膨胀
  - 乾元重宝铸量较大，版别繁多，有小平、折二、折三、折五、折十多种规格
  - 上元元年（公元760年）减重贬值，宝应元年（公元762年）进一步调整币制
- **版别体系**：
  - **小平钱**：标准规格，面文"乾元重宝"，光背为主
  - **折二**：直径较大，当二枚开元通宝
  - **折三**：直径约3.0—3.3厘米
  - **折五**：直径约3.3—3.6厘米
  - **折十**：直径约3.6—4.2厘米，初铸当十
  - **背星月**：背面铸有星纹或月纹
  - **背祥云**：背面铸有祥云纹饰
  - **大字版**：面文字体较大
  - **小字版**：面文字体较小
  - **阔缘版**：外廓较宽
  - **铁钱**：晚期以铁铸造，出土于四川地区
  - **铅钱**：以铅铸造，民间私铸

## 2.4 重轮乾元重宝

### 重轮乾元重宝

- **铸造时间**：唐肃宗乾元二年（公元759年）始铸。据《中国钱币大辞典》记载，由侍御史第五琦奏请铸行重轮大钱，以一当五十枚开元通宝，引发严重通货膨胀。

## 2.11 西域铸币

### 西域铸币

- **铸造时间**：唐代（公元7世纪—9世纪）。据《中国钱币大辞典》记载，唐代西域地区（今新疆一带）除铸行大历元宝、建中通宝外，安西都护府及地方驻军还仿铸开元通宝，以供军需及民间流通使用。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—75%，锡含量约5%—10%，铅含量约10%—20%。唐代铜质总体较好，早期部分含白铜（铜镍合金），晚期因铜材匮乏出现铁钱、铅钱。青铜钱出土后表面多呈青绿色或黑漆古锈蚀，原始铸造色泽为金黄色；白铜钱呈银白色；铁钱呈黑褐色。
- **尺寸重量**：
  - 直径：2.2—2.5厘米
  - 厚度：0.08—0.15厘米
  - 单枚重量：2.0—4.0克
- **面背特征**：
  - 面文多为仿"开元通宝"，但字体简省、变异，笔画粗率
  - 部分钱币面文仅铸"元"字或"开"字等单字
  - 部分钱币面文模糊不清，仅具开元通宝之大概轮廓
  - 外廓不规整，好廓模糊
  - 背面：光背为主
  - 铸工粗率，钱体厚薄不均
- **铸造工艺**：采用范铸法（泥范），一范多钱。西域铸币为唐代（公元7世纪—9世纪）安西都护府及地方驻军就地仿铸开元通宝，以供军需及民间流通使用。制作流程为：先以泥范仿刻开元通宝阴文母范（面文简省或变异），中央方孔设泥芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。因西域远离中原，铸币条件艰苦，铸工粗率，钱体厚薄不均，面文简省或变异，部分钱币面文仅铸单字，具有鲜明的地方特色。

  - 这类仿铸钱币主要出土于新疆库车（龟兹故地）、吐鲁番（高昌故地）地区
  - 西域铸币反映了唐代边疆地区在特殊历史条件下的货币自给现象，具有重要的历史研究价值
  - 与标准开元通宝相比，西域铸币铸工粗糙、面文简省、重量偏轻，具有鲜明的地方特色
