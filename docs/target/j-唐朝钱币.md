# 2. 钱币详情

## 2.1 开元通宝

### 基础信息

- **铸造时间**：唐高祖武德四年（公元621年）始铸，贯穿整个唐代，并延续至五代十国及宋初。据《中国钱币大辞典》记载，开元通宝是中国历史上铸行时间最长的钱币之一，前后流通近三百年。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—75%，锡含量约5%—10%，铅含量约10%—20%。唐代早期部分开元通宝含白铜（铜镍合金），呈银白色；晚期因铜材匮乏出现铁钱、铅钱。另有特殊材质如鎏金钱、银钱等。青铜钱出土后表面多呈青绿色或黑漆古锈蚀，原始铸造色泽为金黄色。
- **尺寸重量**：

- 直径：2.4—2.5厘米
- 厚度：0.1—0.15厘米
- 单枚重量：3.5—4.0克（标准重量为一钱，约3.73克）

### 面背特征

**通用特征**

- 形制：圆形方孔，有内外廓。唐代标准铜钱形制，铸行近三百年
- 铸造工艺：翻砂法，一砂箱多钱，铸工精整
- 青铜质地，铜质精良，铸工规范，为唐代铸币代表
- 开创"通宝"钱制，废止了自秦汉以来的"铢两"钱制，是中国货币史上的重大变革

**面（正面）特征**

- 面文"开元通宝"，四字对读（上下左右）或旋读
- 文字由书法家欧阳询书写，楷书含隶意，端庄雄浑
- "开元"并非年号，而是"开创新纪元"之意
- 外廓规整，好廓清晰

**背（反面）特征**

- 早期光背无文，后期出现星月纹、地名

### 铸造工艺

采用范铸法（铜范或砂范），一范多钱。开元通宝始铸于唐高祖武德四年（公元621年），面文由书法家欧阳询书写，楷书含隶意。制作流程为：先雕刻开元通宝阴文母范（含面文"开元通宝"四字），有外廓和好廓，中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。唐代早期铸工精整，规格统一，标准重量为一钱（约3.73克）。会昌年间（公元845年）毁佛铸钱，各州设钱监铸行，背面铸州名以示铸地，共发现背字二十三种。晚期铸工渐粗，出现减重现象。开元通宝是中国历史上首次采用"通宝"钱制的钱币，终结了沿用七百余年的五铢钱体制。

### 核心背景

- 开元通宝是中国货币史上划时代的里程碑，终结了自汉武帝以来沿行七百余年的"五铢"钱体制
- 开创了以"通宝""元宝"为名的钱币命名制度（通宝钱制），影响直至清末
- "钱"作为重量单位（一两的十分之一）即源于开元通宝的重量标准
- 铸行近三百年，版别极其复杂，是中国古钱币中版别最多的品种之一

## 2.11 西域铸币

### 基础信息

- **铸造时间**：唐代（公元7世纪—9世纪）。据《中国钱币大辞典》记载，唐代西域地区（今新疆一带）除铸行大历元宝、建中通宝外，安西都护府及地方驻军还仿铸开元通宝，以供军需及民间流通使用。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—75%，锡含量约5%—10%，铅含量约10%—20%。唐代铜质总体较好，早期部分含白铜（铜镍合金），晚期因铜材匮乏出现铁钱、铅钱。青铜钱出土后表面多呈青绿色或黑漆古锈蚀，原始铸造色泽为金黄色；白铜钱呈银白色；铁钱呈黑褐色。
- **尺寸重量**：

- 直径：2.2—2.5厘米
- 厚度：0.08—0.15厘米
- 单枚重量：2.0—4.0克

### 面背特征

**通用特征**

- 形制：圆形方孔，有内外廓，钱体较薄
- 铸造工艺：翻砂法，铸工粗率
- 青铜质地，铜质精良，铜质参差
- 唐代西域地方仿铸开元通宝，主要流通于新疆地区

**面（正面）特征**

- 面文多为仿"开元通宝"，但字体简省、变异，笔画粗率
- 部分钱币面文仅铸"元"字或"开"字等单字
- 部分钱币面文模糊不清，仅具开元通宝之大概轮廓
- 外廓不规整，好廓模糊
- 铸工粗率，钱体厚薄不均

**背（反面）特征**

- 光背为主

### 铸造工艺

采用范铸法（泥范），一范多钱。西域铸币为唐代（公元7世纪—9世纪）安西都护府及地方驻军就地仿铸开元通宝，以供军需及民间流通使用。制作流程为：先以泥范仿刻开元通宝阴文母范（面文简省或变异），中央方孔设泥芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。因西域远离中原，铸币条件艰苦，铸工粗率，钱体厚薄不均，面文简省或变异，部分钱币面文仅铸单字，具有鲜明的地方特色。

### 核心背景

- 安史之乱（公元755年）后，河西走廊被吐蕃切断，西域安西都护府、北庭都护府与唐朝中央失去联系
- 西域驻军在失去中央钱币供给的情况下，就地取材仿铸开元通宝，以维持驻军开支和民间贸易
- 这类仿铸钱币主要出土于新疆库车（龟兹故地）、吐鲁番（高昌故地）地区
- 西域铸币反映了唐代边疆地区在特殊历史条件下的货币自给现象，具有重要的历史研究价值
- 与标准开元通宝相比，西域铸币铸工粗糙、面文简省、重量偏轻，具有鲜明的地方特色
