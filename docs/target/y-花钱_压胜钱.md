# 2. 花钱（压胜钱）详细介绍

## 2.1 汉代花钱

### 汉代花钱

* **铸造时间**：西汉至东汉（约公元前206年—公元220年）。

* **材质成分**：青铜。

* **尺寸重量**：直径2.5—5.0厘米不等，大小不一。

* **面背特征**：

  * 圆形方孔或圆形无孔

  * 面文铸吉语（如"长乐未央""常乐""千秋万岁"）

  * 背面铸纹饰（如四神、星象、人物）

  * 铸工古朴

* **核心背景**：

  * 汉代是中国花钱的起源时期

  * 花钱（压胜钱）非流通货币，用于辟邪、祈福、装饰

  * 汉代花钱多铸有吉语，反映了汉代人的思想信仰

  * 据《中国钱币大辞典》记载，汉代花钱为中国花钱的鼻祖

* **版别体系**：

  * **吉语类**：面文铸"长乐未央""常乐""千秋万岁"

  * **四神类**：背面铸青龙、白虎、朱雀、玄武

  * **星象类**：背面铸星象图
