# 2. 三国钱币详细介绍

## 2.1 魏五铢

### 魏五铢

- **铸造时间**：曹魏时期（约公元220年—265年）。据《中国钱币大辞典》记载，曹魏沿用汉代五铢钱制，未铸新钱，沿用旧五铢流通。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约55%—70%，锡含量约5%—10%，铅含量约10%—25%。三国时期因战乱铜材匮乏，铅含量普遍偏高。出土后表面多呈青绿色或灰绿色锈蚀，原始铸造色泽偏暗黄。
- **尺寸重量**：
  - 直径：2.4—2.6cm
  - 厚度：0.08—0.12cm
  - 单枚重量：2.5—3.5g
- **面背特征**：
  - 形制：圆形方孔，有外廓
  - 面文："五铢"，文字风格承袭汉代
  - 铸工较汉代五铢稍粗
  - 背面：有外廓
- **铸造工艺**：采用范铸法（铜范或泥范），一范多钱。魏五铢为曹魏时期沿用汉代五铢钱制铸造，未做重大工艺革新。制作流程为：先雕刻五铢钱阴文母范（含面文"五铢"二字，文字风格承袭汉代），有外廓，中央方孔设泥芯或金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。铸工较汉代五铢稍粗，反映了三国时期战乱频繁、铜材匮乏、铸币工艺水平下降的实际情况。三国时期铅含量普遍偏高，钱币色泽偏暗黄。
- **核心背景**：
  - 魏明帝太和元年（公元227年）恢复铸行五铢钱
  - 魏文帝曹丕曾下令恢复五铢钱的使用，但因谷帛交换盛行，效果有限
  - 曹魏五铢与汉代五铢在形制上较为接近，不易区分
  - 曹魏时期实物货币（谷帛）与铜钱并行，铜钱使用不如汉代普遍
- **版别体系**
  - **标准型魏五铢**：直径2.4—2.6cm，铸工尚可
  - **粗铸型**：铸工较粗糙
- **价值参考**
  - 普品：20—50元
  - 美品（文字清晰）：50—300元

## 2.2 蜀五铢

### 蜀五铢

- **铸造时间**：蜀汉时期（约公元214年—263年）。据《中国钱币大辞典》记载，蜀汉沿行五铢钱制，在四川地区铸行五铢钱。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约55%—70%，锡含量约5%—10%，铅含量约10%—25%。三国时期因战乱铜材匮乏，铅含量普遍偏高。出土后表面多呈青绿色或灰绿色锈蚀，原始铸造色泽偏暗黄。
- **尺寸重量**：
  - 直径：2.1—2.4cm
  - 厚度：0.08—0.12cm
  - 单枚重量：2.0—3.0g
- **面背特征**：
  - 形制：圆形方孔，有外廓
  - 面文："五铢"，"五"字交笔较直，字体较汉代五铢略瘦
  - 铸工尚可
  - 背面：有外廓
- **铸造工艺**：采用范铸法（铜范或泥范），一范多钱。蜀五铢为刘备入蜀后沿用的五铢钱制，形制较汉代五铢略小。制作流程为：先雕刻五铢钱阴文母范（含面文"五铢"二字，"五"字交笔较直，字体较汉代略瘦），有外廓，中央方孔设泥芯或金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。蜀汉因地处西南，铜材来源有限，铸工尚可但不及汉代精整。
- **核心背景**：
  - 蜀汉在四川地区铸行五铢钱，铸工较汉代五铢稍粗
  - 蜀五铢的"五"字交笔较直，与汉代五铢的"五"字交笔弯曲形成对比
  - 蜀汉因连年北伐，财政紧张，同时铸行多种虚值大钱
  - 蜀五铢是蜀汉基础货币，与虚值大钱并行流通
- **版别体系**
  - **标准型蜀五铢**：直径2.1—2.4cm
  - **小型蜀五铢**：直径2.1cm以下
- **价值参考**
  - 普品：30—80元
  - 美品（文字清晰）：80—500元

## 2.3 直百五铢

### 直百五铢

- **铸造时间**：蜀汉建安十九年（公元214年）。据《三国志·蜀书》记载，刘备入蜀后铸行直百五铢，以解决军费不足的问题。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约55%—70%，锡含量约5%—10%，铅含量约10%—25%。三国时期因战乱铜材匮乏，铅含量普遍偏高。出土后表面多呈青绿色或灰绿色锈蚀，原始铸造色泽偏暗黄。
- **尺寸重量**：
  - 直径：2.4—2.8cm
  - 厚度：0.1—0.15cm
  - 单枚重量：3—7g
- **面背特征**：
  - 形制：圆形方孔，有外廓
  - 面文："直百五铢"，四字旋读
  - "直百"即值一百枚五铢钱
  - 背面：部分铸有"为"字（犍为郡铸行），部分光背
  - 背"为"字者为四川犍为郡铸行，是直百五铢中最具辨识性的版别
  - 铸工尚精
- **铸造工艺**：采用范铸法（铜范或泥范），一范多钱。直百五铢为刘备入蜀后（约建安十九年，公元214年）铸行的虚值大钱，面值当百枚五铢。制作流程为：先雕刻直百五铢阴文母范（含面文"直百五铢"四字），有外廓和好廓，中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。部分直百五铢背面铸有"为"字，为四川犍为郡铸行，需在背范中加刻"为"字。铸工尚精。据《三国志·蜀书》记载，刘备"取蜀城中金银铜锡以充军资，铸直百五铢"。
- **核心背景**：
  - 直百五铢是刘备入蜀后铸行的虚值大钱，一枚当一百枚五铢钱
  - 刘备初入蜀时，军费匮乏，接受刘巴建议铸行直百五铢，数月之间府库充实
  - 直百五铢的铸行虽解决了短期财政困难，但导致了严重的通货膨胀
  - 背"为"字者为犍为郡铸行，是三国钱币中罕见的纪地钱币
  - 直百五铢后期减重严重，由初铸时的7g减至3g以下
- **版别体系**
  - **初铸厚重型**：直径2.6—2.8cm，重5—7g，铸工较精
  - **标准型**：直径2.4—2.6cm，重3—5g
  - **减重型**：直径2.4cm以下，重3g以下
  - **背"为"字版**：背面穿左铸"为"字，犍为郡铸行
  - **背星纹版**：背面铸有星纹
  - **背阴纹版**：背面铸有阴刻纹饰
- **价值参考**
  - 普品（光背）：100—300元
  - 美品（背"为"字）：500—3,000元
