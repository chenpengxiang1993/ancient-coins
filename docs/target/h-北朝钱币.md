# 2. 北朝钱币详细介绍

## 2.1 太和五铢

### 太和五铢

- **铸造时间**：北魏太和十九年（公元495年）。据《魏书·食货志》记载，孝文帝迁都洛阳后铸行太和五铢，为北魏首次铸币。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约55%—70%，锡含量约5%—15%，铅含量约10%—25%。北朝各政权铜质参差。出土后表面多呈青绿色锈蚀，原始铸造色泽偏暗黄至金黄色。
- **尺寸重量**：
  - 直径：约2.4—2.6cm
  - 厚度：0.08—0.12cm
  - 单枚重量：约2—3.5g
- **面背特征**：
  - 形制：圆形方孔，有外廓
  - 面文："太和五铢"，四字对读
  - "太和"为孝文帝年号
  - 文字为篆书，笔画古朴
  - 铸工较粗，规格不一
  - 背面：平素无文
- **铸造工艺**：采用范铸法（铜范或泥范），一范多钱。太和五铢为北魏孝文帝太和十九年（公元495年）铸行，是北魏首次铸行的钱币。制作流程为：先雕刻太和五铢阴文母范（含面文"太和五铢"四字，篆书），中央方孔设泥芯或金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。铸工较粗，规格不一，反映了北魏初期铸币工艺尚处于发展阶段。
- **核心背景**：
  - 太和五铢是北魏首次铸行的钱币，标志着鲜卑族政权从实物经济向货币经济的转变
  - 孝文帝迁都洛阳后推行汉化改革，铸行太和五铢是汉化改革的重要组成部分
  - 太和五铢铸量不多，流通范围有限，民间仍以布帛为主要交易媒介
  - 太和五铢是研究北魏汉化改革的重要实物
- **版别体系**
  - **标准型太和五铢**：直径2.4—2.6cm
  - **大字版**：面文字体较大
  - **小字版**：面文字体较小
- **价值参考**
  - 普品：200—500元
  - 美品（文字清晰）：500—3,000元
