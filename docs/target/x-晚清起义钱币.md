# 2. 晚清起义钱币详细介绍

## 2.1 平靖通宝

### 平靖通宝

* **铸造时间**：清咸丰五年（公元1855年），天地会所铸。

* **材质成分**：黄铜（铜锌合金）或青铜（铜锡铅合金），铜含量约55%—65%。晚清起义政权铸币铜质参差。黄铜钱呈金黄色，青铜钱表面多呈青绿色锈蚀。

* **尺寸重量**：小平直径约2.4厘米，重约3—3.5克。

* **面背特征**：

  * 圆形方孔

  * 面文"平靖通宝"四字楷书直读

  * 背面铸"中"字

  * 铸工尚精

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。平靖通宝为天地会（洪门）于清咸丰五年（公元1855年）在广东、广西起义时铸行。制作流程为：先雕刻平靖通宝阴文母范（含面文"平靖通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融黄铜液或青铜液，冷却脱范后逐枚修整。铜质参差，铸工尚精，背面铸"中""前""后"等字代表营制。

* **核心背景**：

  * 天地会（洪门）于咸丰年间在广东、广西起义时铸行

  * "平靖"意为平定安定

  * 背"中"字代表"中营"

  * 据《中国钱币大辞典》记载，平靖通宝为晚清天地会铸币的重要品种

* **版别体系**：

  * **小平背"中"版**：背面铸"中"字

  * **小平背"前"版**：背面铸"前"字

  * **小平背"后"版**：背面铸"后"字

## 2.2 平靖胜宝

### 平靖胜宝

* **铸造时间**：清咸丰年间（公元1855年前后），天地会所铸。

* **材质成分**：黄铜（铜锌合金）或青铜（铜锡铅合金），铜含量约55%—65%。晚清起义政权铸币铜质参差。黄铜钱呈金黄色，青铜钱表面多呈青绿色锈蚀。

* **尺寸重量**：大钱型制，具体尺寸重量因存世极罕而缺少系统记载。

* **面背特征**：

  * 圆形方孔

  * 面文"平靖胜宝"四字楷书直读

  * 光背或背有文字

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。平靖胜宝为天地会于清咸丰年间（约公元1855年前后）铸行的赏功钱，非流通货币。制作流程为：先雕刻平靖胜宝阴文母范（含面文"平靖胜宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融金液、银液或铜液，冷却脱范后逐枚修整。分金、银、铜三等，用于赏赐有功会众，存世极罕。

* **核心背景**：

  * 平靖胜宝为天地会赏功钱性质，非流通货币

  * 分金、银、铜三等，用于赏赐有功会众

  * 存世极罕，与明末张献忠"西王赏功"钱性质类似

  * 据《中国钱币大辞典》记载，平靖胜宝为晚清钱币中的珍品

* **版别体系**：

  * **金质版**：黄金铸造，存世极罕

  * **银质版**：白银铸造

  * **铜质版**：青铜铸造
