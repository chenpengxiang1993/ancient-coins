# 2. 钱币详情

## 2.1 大泉五十

### 基础信息

- **铸造时间**：王莽居摄二年（公元7年）始铸，始建国元年（公元9年）大量铸行，贯穿整个新莽时期。据《汉书·食货志》记载，大泉五十是新莽时期铸行量最大的钱币。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约65%—80%，锡含量约5%—15%，铅含量约5%—15%。据《中国古钱谱》分析，新莽钱币铜质精良，铸工极精。出土后表面多呈青绿色或黑漆古锈蚀，原始铸造色泽为明亮金黄色。
- **尺寸重量**：

- 直径：2.7—4.0cm（大小不一，后期渐小）
- 厚度：0.15—0.25cm
- 单枚重量：3—10g

### 面背特征

**通用特征**

- 形制：圆形方孔，有外廓和好廓
- 文字为悬针篆，笔画纤细如针，优雅工整
- 外廓高而规整，好廓清晰

**面（正面）特征**

- "大泉五十"，四字对读（上下左右）

**背（反面）特征**

- 平素无文

### 铸造工艺

采用母范翻铸法（铜母范翻制泥子范），一范多钱，为新莽铸币的标准工艺。大泉五十为王莽始建国元年（公元9年）第一次币制改革时铸行。制作流程为：先以高精度铜范雕刻"大泉五十"阴文母范（含面文四字，悬针篆书体，笔画纤细如针），外廓高而规整，好廓清晰，中央方孔设金属芯；以铜母范翻制多枚泥质子范，子范干透后合范浇铸熔融青铜液，冷却脱范后逐枚修整打磨。新莽铸币以工艺精良著称，母范翻铸法使钱文更加规整统一，铸工远超西汉五铢。据《汉书·食货志》记载："莽即真，以为刘字有卯金刀，乃罢错刀、契刀及五铢钱，更作大泉五十。"

### 核心背景

- "大泉五十"意为该铜泉（钱）值五十枚五铢钱，是虚值大钱
- 是新莽时期铸行量最大的钱币，流通时间最长
- 悬针篆书体为新莽钱币的标志性书法，笔画纤细匀称，极具艺术价值
- 大泉五十铸工精整，是新莽钱币的代表作
- 后期出现减重现象，直径和重量逐渐减小

- 东汉初年仍沿用货泉，直至建武十六年（公元40年）恢复五铢钱制
- 货泉的铸行说明王莽最终回归了接近五铢钱的重量标准
- 存世量极大，是新莽钱币中最常见的品种

## 2.21 布泉-新莽

### 基础信息

- **铸造时间**：王莽天凤年间（公元14年—19年）。据《中国钱币大辞典》记载，新莽布泉与北周布泉同名但年代不同，需注意区分。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约65%—80%，锡含量约5%—15%，铅含量约5%—15%。据《中国古钱谱》分析，新莽钱币铜质精良，铸工极精。出土后表面多呈青绿色或黑漆古锈蚀，原始铸造色泽为明亮金黄色。
- **尺寸重量**：

- 直径：约2.5—2.6cm
- 厚度：0.1—0.15cm
- 单枚重量：约3—4g

### 面背特征

**通用特征**

- 形制：圆形方孔，有外廓和好廓
- 外廓规整

**面（正面）特征**

- "布泉"，悬针篆，铸工极精，文字优美

**背（反面）特征**

- 平素无文

### 铸造工艺

采用母范翻铸法（铜母范翻制泥子范），一范多钱。布泉-新莽为王莽时期铸行，铸工极精，文字优美，悬针篆书体堪称莽钱书法精品。制作流程为：先以高精度铜范雕刻"布泉"阴文母范（含面文二字，悬针篆），有外廓和好廓，中央方孔设金属芯；以铜母范翻制泥质子范，合范浇铸熔融青铜液，冷却脱范后逐枚精心修整。布泉铸工极精，外廓规整，为新莽铸币工艺的代表作品。需注意与北周布泉区分（新莽布泉为悬针篆，北周布泉为玉箸篆）。

### 核心背景

- 布泉-新莽为王莽时期铸行的方孔圆钱
- 与北周布泉同名但年代不同：新莽布泉为悬针篆，北周布泉为玉箸篆
- 新莽布泉的铸行背景尚有争议，一说为天凤年间铸行
- 存世量较货泉少，但仍可常见

## 2.22 国宝金匮直万

### 基础信息

- **铸造时间**：新莽时期（约公元9年—14年）。据《中国钱币大辞典》记载，国宝金匮直万是中国古钱币中最为珍稀的品种之一。
- **材质成分**：青铜（铜、锡、铅合金），铜含量约70%—80%，铜质精良。出土后表面多呈青绿色或黑漆古锈蚀，原始铸造色泽为明亮金黄色。
- **尺寸重量**：

- 上部（方孔圆钱部分）：直径约3.0cm
- 下部（方形部分）：约3.0cm×2.0cm
- 全长：约6.0cm
- 单枚重量：约15—30g

### 面背特征

**通用特征**

- 形制：上部为方孔圆钱，面文"国宝金匮"，下部为方形台座，铸"直万"二字。整体造型独特，为王莽时期最珍稀的铸币之一

**面（正面）特征**

- 形制独特：上部为方孔圆钱，铸"国宝金匮"四字；下部为方形，铸"直万"二字
- 文字为悬针篆
- 铸工尚精

**背（反面）特征**

- 平素无文

### 铸造工艺

采用母范翻铸法（铜母范翻制泥子范），一范一钱。国宝金匮直万形制极为独特，上部为方孔圆钱铸"国宝金匮"四字，下部为方形铸"直万"二字，整体呈上圆下方形制，范腔雕刻工艺极为复杂。制作流程为：先以铜范雕刻国宝金匮直万阴文母范（含上部方孔圆钱和下部方形两部分的完整形态），需在范腔中精确表现上圆下方的过渡连接，中央方孔设泥芯或金属芯；以铜母范翻制泥质子范，合范浇铸熔融青铜液，冷却脱范后精心修整。铸工尚精，文字为悬针篆。国宝金匮直万铸量极少，存世极罕，名列古泉五十名珍之首。据《汉书·食货志》记载，王莽多次改革币制，此钱为其中最特殊者。

### 核心背景

- 国宝金匮直万是中国古钱币中形制最为独特的品种之一，上部圆钱下部方形的形制独一无二
- "直万"意为值一万枚五铢钱，是中国历史上面值最大的铜钱
- 存世极罕，已知仅数枚，均为博物馆或顶级私人收藏
- 国宝金匮直万位列古泉五十名珍之首，是中国古钱币收藏的最高目标之一
- 关于其铸行背景，学术界尚有争议，一说为王莽宝货制中的顶级货币
- 中国国家博物馆藏有一枚，为该馆镇馆之宝之一
