# 2. 三藩钱币详细介绍

## 2.1 利用通宝

### 利用通宝

* **铸造时间**：清康熙十三年（公元1674年），吴三桂起兵时所铸。

* **材质成分**：黄铜（铜锌合金），铜含量约55%—65%，锌含量约25%—35%。吴三桂铸币。黄铜钱呈金黄色，表面多呈深褐色包浆。

* **尺寸重量**：

  * 小平直径：约2.5厘米，重约3—3.5克

  * 折二直径：约2.8厘米

  * 折五直径：约3.3厘米

  * 折十直径：约4.0厘米

* **面背特征**：

  * 圆形方孔

  * 面文"利用通宝"四字楷书直读

  * 有光背及背纪值（如"一分""一钱"）

  * 铸于云南

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。利用通宝为吴三桂于清康熙十三年（公元1674年）起兵叛清后铸行。制作流程为：先雕刻利用通宝阴文母范（含面文"利用通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融黄铜液，冷却脱范后逐枚修整。铸于云南，在云南、贵州、湖南一带流通，铸行时间较长。

* **核心背景**：

  * 清康熙十二年（公元1673年）朝廷下令撤藩，吴三桂起兵叛清，自称"天下都招讨兵马大元帅"

  * 次年铸行"利用通宝"，在云南、贵州、湖南一带流通

  * 三藩之乱历时八年，利用通宝铸行时间较长

  * 据《中国钱币大辞典》记载，利用通宝为三藩钱币中最常见的品种

* **版别体系**：

  * **小平光背版**：背面无文

  * **折二版**：较少见

  * **折五背"一分"版**：背面铸"一分"

  * **折十背"一钱"版**：背面铸"一钱"

## 2.2 昭武通宝

### 昭武通宝

* **铸造时间**：清康熙十七年（公元1678年），吴三桂所铸。

* **材质成分**：黄铜（铜锌合金），铜含量约55%—65%，锌含量约25%—35%。吴三桂铸币。黄铜钱呈金黄色，表面多呈深褐色包浆。

* **尺寸重量**：

  * 小平直径：约2.5厘米

  * 折十直径：约3.5厘米

* **面背特征**：

  * 圆形方孔

  * 有楷书、篆书两种书体

  * 楷书版直读

  * 篆书版较少见

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。昭武通宝为吴三桂于康熙十七年（公元1678年）在衡阳称帝后铸行，国号"周"，建元"昭武"。制作流程为：先雕刻昭武通宝阴文母范（含面文"昭武通宝"，楷书或篆书），中央方孔设金属芯；合范浇铸熔融黄铜液，冷却脱范后逐枚修整。有楷书、篆书两种书体。吴三桂称帝后不久即病死，铸期极短，存世较少。

* **核心背景**：

  * 吴三桂于康熙十七年（公元1678年）在衡阳称帝，国号"周"，建元"昭武"，铸行昭武通宝

  * 吴三桂称帝后不久即病死，昭武年号仅使用数月

  * 昭武通宝铸期极短，存世较少

  * 据《中国钱币大辞典》记载，昭武通宝篆书版尤为珍罕

* **版别体系**：

  * **楷书小平版**：面文楷书，光背

  * **篆书小平版**：面文篆书，较少见

  * **折十版**：较少见

## 2.3 洪化通宝

### 洪化通宝

* **铸造时间**：清康熙十七年（公元1678年）至二十年（公元1681年），吴世璠所铸。

* **材质成分**：黄铜（铜锌合金），铜含量约55%—65%，锌含量约25%—35%。吴世璠铸币。黄铜钱呈金黄色，表面多呈深褐色包浆。

* **尺寸重量**：小平直径约2.5厘米，重约3—3.5克。

* **面背特征**：

  * 圆形方孔

  * 面文"洪化通宝"四字楷书直读

  * 光背或背纪局

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。洪化通宝为吴三桂之孙吴世璠继位后改元"洪化"时铸行（公元1678年—1681年）。制作流程为：先雕刻洪化通宝阴文母范（含面文"洪化通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融黄铜液，冷却脱范后逐枚修整。铸行约四年，至康熙二十年清军攻破昆明，三藩之乱平定。

* **核心背景**：

  * 吴三桂死后，其孙吴世璠继位，改元"洪化"，铸行洪化通宝

  * 洪化通宝铸行约四年，至康熙二十年（公元1681年）清军攻破昆明，三藩之乱平定

  * 据《中国钱币大辞典》记载，洪化通宝为三藩钱币中的常见品种

* **版别体系**：

  * **小平光背版**：背面无文

  * **小平背纪局版**：背面铸纪局文字

## 2.4 裕民通宝

### 裕民通宝

* **铸造时间**：清康熙十五年（公元1676年），耿精忠所铸。

* **材质成分**：黄铜（铜锌合金），铜含量约55%—65%，锌含量约25%—35%。耿精忠铸币。黄铜钱呈金黄色，表面多呈深褐色包浆。

* **尺寸重量**：

  * 小平直径：约2.5厘米

  * 折一大型直径：约2.8厘米，背"一分"

  * 折十型直径：约4.0厘米，背"一钱"

* **面背特征**：

  * 圆形方孔

  * 面文"裕民通宝"四字楷书直读

  * 背面铸"一分""一钱"纪值文字

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。裕民通宝为靖南王耿精忠于清康熙十五年（公元1676年）据福建响应吴三桂叛清后铸行。制作流程为：先雕刻裕民通宝阴文母范（含面文"裕民通宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融黄铜液，冷却脱范后逐枚修整。背面铸"一分""一钱"纪值文字，意为"富裕百姓"。耿精忠后降清，铸期较短。

* **核心背景**：

  * 耿精忠为靖南王，据福建响应吴三桂叛清

  * 耿精忠在福建铸行裕民通宝，意为"富裕百姓"

  * 耿精忠后降清，裕民通宝铸期较短

  * 据《中国钱币大辞典》记载，裕民通宝存世量有限

* **版别体系**：

  * **小平光背版**：背面无文。面文"裕民通宝"楷书笔画端庄

  * **折一背"一分"版**：背面铸"一分"二字，笔画端正

  * **折十背"一钱"版**：背面铸"一钱"二字，钱体厚重饱满

  * **浙版**：铸于浙江，面文风格略有差异，字体较粗犷
