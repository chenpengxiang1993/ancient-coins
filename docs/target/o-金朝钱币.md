# 2. 金朝钱币详细介绍

## 2.1 天辅元宝

### 天辅元宝

* **铸造时间**：金太祖完颜阿骨打天辅年间（公元1117年—1123年）。

* **材质成分**：青铜（铜、锡、铅合金），铜含量约60%—70%，锡含量约5%—10%，铅含量约15%—25%。金代铜钱铸工较精。出土后表面多呈青绿色锈蚀，原始铸造色泽为金黄色。

* **尺寸重量**：直径约2.4厘米，单枚重量约3.5克。

* **面背特征**：

  * ***面特征***

    * 面文"天辅元宝"，四字旋读，楷书

  * ***背特征***

    * 背面光素无文

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。天辅元宝为金太祖完颜阿骨打天辅年间（公元1117年—1123年）铸行，为金朝最早铸币。制作流程为：先雕刻天辅元宝阴文母范（含面文"天辅元宝"四字，楷书），中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。金初铸币技术尚在初创阶段，铸量极少，存世极罕。

* **核心背景**：天辅元宝为金最早铸币，存世极罕。马定祥十级制定为三级（罕贵）。

* **版别体系**：仅见小平钱。

## 2.8 大定通宝

  * 单枚重量：约3.5—4.0克

* **面背特征**：


  * 铸工极精，文字规整，仿宋徽宗瘦金体风格


  * ***面特征***

    * 面文"大定通宝"，四字对读，楷书

  * ***背特征***

    * 背面有光背、背"申"、背"酉"（纪年）

* **铸造工艺**：采用范铸法（铜范或砂范），一范多钱。大定通宝为金世宗完颜雍大定年间（公元1161年—1189年）铸行，为金代铸工最精的钱币。制作流程为：先雕刻大定通宝阴文母范（含面文"大定通宝"四字，楷书，仿宋徽宗瘦金体风格），中央方孔设金属芯；合范浇铸熔融青铜液，冷却脱范后逐枚修整。铸工极精，文字规整，背面有光背及纪年（背"申"、背"酉"）两种。另有铁母、白铜版等特殊铸造品种。

* **核心背景**：

  * 大定通宝为金代铸工最精的钱币，世宗被誉为"小尧舜"，大定年间政治清明

  * 铸工之精可与北宋大观通宝媲美，为金代钱币中的精品

  * 马定祥十级制定为八级（少）

* **版别体系**：

  * **按背面文字分**：据《中国钱币大辞典》记载，大定通宝背面有光背和纪年两种

    * **背"申"版**：背面穿下铸"申"字，笔画端正清晰（大定二十八年，戊申年，公元1188年铸）
    * **背"酉"版**：背面穿下铸"酉"字，笔画端正清晰（大定二十九年，己酉年，公元1189年铸）

  * **按面文风格分**：
    * **大字版**：面文"大定通宝"字体较大，笔画粗壮，字口深峻。较少见
    * **小字版**：面文字体较小
    * **阔缘版**：外缘较宽
    * **窄缘版**：外缘较窄
    * **瘦金体版**：面文仿宋徽宗瘦金体风格，笔画纤细挺拔

  * **特殊版别**：
    * **铁母**：铁钱的母钱，铜质精良，极为珍贵
    * **白铜版**：白铜铸造，铜色银白，较为少见
    * **母钱**：铸工精整，字口深峻
